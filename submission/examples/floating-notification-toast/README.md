# Floating Notification Toast

## What it demonstrates
- A floating toast stack (bottom-right)
- Four variants: **success**, **error**, **warning**, **info**
- Slide-in entrance + fade-out dismissal
- Hover pause effect for the auto-dismiss progress bar

## How it works (brief)
- `demo.html` contains the minimal JS to create/dismiss toasts and auto-dismiss them on a timer.
- `style.css` defines the toast layout + animations (`fn-slide-in` and `fn-fade-out`).
- The progress bar is an animated element; hovering a toast pauses/resumes the progress via `animation-play-state`.
