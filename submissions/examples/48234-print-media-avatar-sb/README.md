# Print-Optimized Avatar Component

Avatar components with print media query optimization. Avatars are hidden when printing to save ink.

## Files

- `demo.html` - Avatar demos with print optimization
- `style.css` - Avatar styling with @media print rules

## Usage

Open `demo.html` in a browser to see the styled avatars. Use Ctrl+P (Cmd+P on Mac) to see avatars hidden in print mode.

## What This Does

Adds print media query optimization to avatar components, hiding all avatars when printing. Avatars are purely decorative visual elements that don't need to appear on printed pages.

## Why Useful

Avatar images would waste ink and may not display well on paper. Hiding them produces cleaner printed output and saves ink.
