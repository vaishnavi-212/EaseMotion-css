# CSS Horizontal Roll Popover for Neumorphic Soft Layouts

A pure CSS **Horizontal Roll Popover** featuring a smooth horizontal roll animation with a soft neumorphic interface. The component is lightweight, responsive, keyboard accessible, and requires no JavaScript.

## Features

- Pure HTML & CSS
- Smooth Horizontal Roll animation
- Neumorphic Soft UI
- Responsive layout
- Keyboard accessible
- CSS Custom Properties
- Lightweight
- No JavaScript
