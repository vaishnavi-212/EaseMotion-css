# Feature Adoption Heatmap

1. What does this do? Adds responsive feature adoption rows with animated heatmap cells, adoption score chips, staggered entry motion, and hover or focus feedback.
2. How is it used? Apply `.heatmap-grid` to the wrapper and `.adoption-row` with tone classes like `.high-row`, `.medium-row`, or `.low-row` for each feature.
3. Why is it useful? It helps teams compare feature usage quickly with purposeful CSS-only motion and readable adoption states.
