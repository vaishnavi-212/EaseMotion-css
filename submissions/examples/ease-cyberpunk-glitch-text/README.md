# Animated Cyberpunk Glitch Text

A high-fidelity pure CSS text glitch animation. Perfect for cyberpunk themes, 404 error pages, or high-energy landing pages.

## Features
- **Pure CSS**: Uses `::before` and `::after` pseudo-elements combined with `clip` and intense keyframes to slice the text in real-time.
- **RGB Splitting**: Utilizes red and cyan text-shadows offset against each other to create a true chromatic aberration effect.
- **Infinite Randomness**: The keyframes are highly randomized and run on alternating linear loops to make the glitch look unpredictable.

## How to use

Make sure the `data-text` attribute matches the inner HTML text exactly.

```html
<h1 class="ease-cyberpunk-glitch" data-text="CYBERPUNK">CYBERPUNK</h1>
```
