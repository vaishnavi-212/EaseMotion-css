# Ease Kubernetes Resource Quota Breakdown

## Overview

A responsive HTML and CSS component for monitoring Kubernetes namespace resource quotas and utilization.

## Features

- CPU quota tracking
- Memory utilization monitoring
- Persistent volume usage
- Running pod quota alerts
- Responsive design
- Pure HTML & CSS
- CSS variables
- Hover effects and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable Kubernetes operations component for monitoring namespace quotas, preventing resource exhaustion, and improving cluster capacity planning.
