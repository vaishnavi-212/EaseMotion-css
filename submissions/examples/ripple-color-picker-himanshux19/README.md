# Ripple Color Picker

A modern **Ripple Color Picker** component built for the EaseMotion CSS library. Inspired by Neon UI design patterns, this component provides an elegant way to select colors using smooth ripple animations, glowing effects, and responsive design—all without JavaScript.

## ✨ Features

- Pure HTML & CSS
- No JavaScript required
- Smooth ripple animation
- Neon-inspired glow effects
- Responsive layout
- Keyboard accessible
- Uses semantic HTML
- Easy to customize
- Lightweight and reusable

## 📁 Structure

```
ripple-color-picker-himanshux19/

└── README.md
```

## 🚀 Usage

Include the stylesheet:

```html
<link rel="stylesheet" href="style.css">
```

Import EaseMotion CSS before this component.

## 🎨 Customization

Modify the CSS variables to match your design.

```css
:root{
    --picker-size:64px;
    --glow-size:18px;
    --transition:.35s;
}
```

Change the swatch colors:

```css
.red{
    --color:#ff4d6d;
}

.blue{
    --color:#4d7dff;
}
```
