# CSS Shimmer Sweep Tooltip for Accessible Web Layouts

A modern tooltip component built using pure HTML and CSS featuring a smooth shimmer sweep animation on hover.

## Features

- Pure HTML & CSS
- Smooth shimmer sweep animation
- Accessible layout
- Responsive design
- Lightweight and reusable

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern web browser.
