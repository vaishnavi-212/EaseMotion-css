# Print-Optimized Button Component

A collection of button components with print media query optimization. Buttons render as simple black-and-white elements when printed.

## Files

- `demo.html` - Button demos with print optimization
- `style.css` - Button styling with @media print rules

## Usage

Open `demo.html` in a browser to see the styled buttons. Use Ctrl+P (Cmd+P on Mac) to see simplified print styling.

## What This Does

Adds print media query optimization to button components, removing backgrounds, shadows, and color styling in print mode. Buttons render as simple bordered elements with black text.

## Why Useful

Buttons with colorful backgrounds and shadows waste ink when printed. This optimization ensures buttons render cleanly as simple bordered elements on paper.
