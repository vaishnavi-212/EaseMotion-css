## Description

Adds a responsive and accessible CSS-only 3D Flip Carousel to the EaseMotion examples library.
The component uses native radio controls and CSS sibling selectors to switch between slides with a smooth 3D `rotateY()` transition.

## Features

- Pure CSS carousel with no JavaScript
- Smooth 3D flip transition
- Keyboard-operable native radio controls
- Responsive desktop, tablet, and mobile layouts
- Visible focus indicators
- No autoplay
- Reduced-motion support
- Forced-colors support
- Configurable CSS custom properties for timing, easing, scale, perspective, spacing, and dimensions

## Accessibility

- Native keyboard-accessible controls
- Descriptive labels for every slide
- `:focus-visible` styling
- `prefers-reduced-motion` support
- Touch-friendly control sizes
- Semantic headings and content structure
