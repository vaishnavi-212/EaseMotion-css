# Elastic Bounce Popover – Dashboard Analytics

A standalone responsive popover showcasing dashboard analytics with an Elastic Bounce animation using pure HTML and CSS.

## Features

- Pure HTML & CSS
- Elastic Bounce animation
- Dashboard analytics interface
- KPI metrics popover
- Responsive layout
- Keyboard accessible
- CSS custom properties
- Zero JavaScript

## Files

```
demo.html
style.css
README.md
```

## Usage

Open `demo.html` directly in a browser.

Customize:

```css
--primary
--success
--duration
--radius
```
