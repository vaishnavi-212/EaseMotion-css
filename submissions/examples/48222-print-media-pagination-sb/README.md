# Print-Optimized Pagination Component

Pagination component with print media query optimization. Pagination is hidden when printing.

## Files

- `demo.html` - Pagination demo with print optimization
- `style.css` - Pagination styling with @media print rules

## Usage

Open `demo.html` in a browser to see the styled pagination. Use Ctrl+P (Cmd+P on Mac) to see pagination hidden in print mode.

## What This Does

Adds print media query optimization to pagination components, hiding them when printing. Pagination controls are interactive elements that don't need to appear on printed pages.

## Why Useful

Pagination is for navigating between pages in digital format. When content is printed, all relevant content should be included. Hiding pagination prevents confusion on printed pages.
