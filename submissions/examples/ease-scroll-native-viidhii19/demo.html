Native CSS Scroll Reveals

Hardware Accelerated

This card fades and slides in purely using the native CSS view() timeline, avoiding the main thread entirely.

Zero JavaScript

No IntersectionObservers, no scroll event listeners, and no scripting overhead. 100% CSS styling.

Scroll-Linked Progression

The animation frame is inextricably linked to the user's scroll depth. Scroll up and down to see it reverse organically.

Graceful Degradation

For browsers that lack support for animation-timeline, a @supports fallback ensures content remains visible and accessible immediately.

Range Calibrated

Using animation-range, we dictate exactly when the transition begins and ends relative to the viewport's intersection bounds.
