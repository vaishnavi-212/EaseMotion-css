# Crystallize Progress Bar

A modern crystallize-inspired progress bar built with pure HTML and CSS using EaseMotion CSS.

## Features

- Crystal-style animated progress effect
- Pure HTML & CSS
- Responsive layout
- Accessible structure
- Uses EaseMotion CSS variables and keyframes
