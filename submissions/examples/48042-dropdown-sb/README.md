# Dropdown Component

A clean dropdown menu component with smooth animations and click-outside support.

## Features

- Smooth open/close animations
- Click outside to close
- Divider support
- Customizable trigger text

## Usage

```html
<div class="dropdown" id="dropdown1">
  <button class="dropdown-trigger" onclick="toggleDropdown('dropdown1')">Select Option</button>
  <div class="dropdown-menu">
    <a href="#" class="dropdown-item">Option 1</a>
    <a href="#" class="dropdown-item">Option 2</a>
  </div>
</div>
```
