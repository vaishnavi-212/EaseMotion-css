# Pipeline Stage Glow

1. What does this do? Adds responsive build pipeline rows with animated glowing progress rails, stage status badges, staggered entry motion, and hover or focus feedback.
2. How is it used? Apply `.stage-list` to the wrapper and `.stage-row` with state classes like `.complete-stage`, `.active-stage`, or `.queued-stage` for each pipeline step.
3. Why is it useful? It helps release dashboards show pipeline progress clearly with purposeful CSS-only motion and keyboard-friendly focus treatment.
