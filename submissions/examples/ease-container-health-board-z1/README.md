# Ease Container Health Board

## Overview
A CSS-only component for monitoring container health status.

## Features
- Animated status badges
- Responsive layout
- Hover effects
- CSS variables
- Pure HTML & CSS

## Usage
Open `demo.html` in any browser.

## Browser Support
Chrome, Firefox, Edge, Safari.

## Why EaseMotion CSS?
A reusable animation-first component for Kubernetes dashboards, Docker monitoring tools, DevOps platforms, and cloud infrastructure panels.
