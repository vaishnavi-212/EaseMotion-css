# Ease Weather Status Card

A reusable Weather Status Card built using pure HTML and CSS.

## Features

- Modern weather card UI
- Floating weather icon animation
- Temperature display
- Humidity and wind statistics
- Responsive layout
- Pure HTML & CSS
- No JavaScript required

## Folder Structure

```
ease-weather-status-card/
├── demo.html
├── style.css
└── README.md
```

## Usage

```html
<link rel="stylesheet" href="style.css">
```

Open `demo.html` directly in your browser.

## Customization

- Change weather icon
- Update temperature values
- Modify colors
- Customize animation speed
- Adjust card size

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
