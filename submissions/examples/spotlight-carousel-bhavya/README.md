# Spotlight Carousel

A pure CSS carousel with radio-button navigation and animated spotlight card effects.

## How to use

Open `demo.html` directly in a browser and click the numbered navigation pills.

```html
<input type="radio" name="slide" id="slide-1" checked />
<div class="viewport">
  <div class="slides">
    <article class="slide">...</article>
  </div>
</div>
<label for="slide-1">01</label>
```

## Why it is useful

Carousels are common in startup landing pages, product highlights, feature showcases, and onboarding sections. This version uses only HTML and CSS while still providing animated depth, responsive layout, and clear navigation.
