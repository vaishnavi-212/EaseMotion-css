# Pricing Card Spotlight

This example adds a focused pricing-card animation that reacts to hover and keyboard focus with a soft spotlight sweep.

## Files

- `demo.html` contains the accessible pricing card markup.
- `style.css` defines the panel styling, spotlight sweep, focus state, and reduced-motion fallback.

## Highlights

- Keyboard-focusable card using `tabindex="0"`.
- Smooth hover and focus-visible spotlight movement.
- Stable layout dimensions so the price, features, and CTA do not shift during the animation.
- `prefers-reduced-motion` support for users who request less motion.
