# Print-Optimized Kbd Component

Keyboard key styling with print media query optimization. Keyboard keys render with simple borders when printed.

## Files

- `demo.html` - Kbd demo with print optimization
- `style.css` - Kbd styling with @media print rules

## Usage

Open `demo.html` in a browser to see the keyboard key styling. Use Ctrl+P (Cmd+P on Mac) to see simplified print styling.

## What This Does

Adds print media query optimization to kbd components, removing shadows and gradients while keeping borders for readability on paper.

## Why Useful

Keyboard keys with 3D shadows and gradients waste ink when printed. This optimization ensures keys render as clean, readable elements.
