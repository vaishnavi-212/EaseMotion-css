# Floating Orbit Tooltip - SaaS Showcase Layouts

A CSS tooltip component with floating orbit animation effect designed for SaaS showcase pages.

## Files

- `demo.html` - Demo page showing SaaS pricing and features with tooltips
- `style.css` - Tooltip styles and animations

## Usage

Open `demo.html` in a browser and hover over buttons to see the floating orbit tooltip animation.

## Features

### Floating Orbit Animation
- Tooltip gently floats with subtle rotation
- Purple/magenta gradient with glow
- Smooth continuous animation loop

### SaaS Components
- Pricing cards with multiple plans
- Feature showcase cards
- CTA buttons with tooltips
- Dark theme design

### Design System
- Dark background SaaS aesthetic
- Purple accent color
- Gradient buttons and icons

## Classes

| Class | Description |
|-------|-------------|
| `ease-floating-orbit-saas-sb` | Floating orbit tooltip for SaaS |
| `ease-saas-btn-sb` | SaaS outline button |
| `ease-saas-btn-primary-sb` | SaaS primary button |
| `ease-saas-learn-btn-sb` | Learn more button |
| `ease-saas-cta-primary-sb` | CTA primary button |
| `ease-saas-cta-secondary-sb` | CTA secondary button |
| `pricing-card` | Pricing plan card |
| `feature-card` | Feature showcase card |

## Usage

Add `data-tooltip="Your text"` attribute to any element with `ease-floating-orbit-saas-sb` class:

```html
<button class="ease-floating-orbit-saas-sb" data-tooltip="Start free trial">
  Get Started
</button>
```
