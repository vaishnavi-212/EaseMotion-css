# Fade In Map Embed

A clean, responsive map embed component designed for corporate websites, company contact pages, and business dashboards. It uses a simple fade-in animation to present the map smoothly without requiring JavaScript.

---

## Features

- Smooth fade-in entrance animation
- Corporate-inspired professional layout
- Pure HTML and CSS (no JavaScript)
- Fully responsive on desktop, tablet, and mobile
- Accessible iframe with descriptive title
- Easy to customize colors, spacing, and content
- Compatible with EaseMotion CSS utility classes

## Usage

Include the component in your HTML:

```html
<section class="corporate-map-card ease-fade-in">

    <div class="company-info">
        ...
    </div>

    <div class="map-container">
        <iframe
            title="Corporate Office Location"
            src="https://maps.google.com/maps?q=New%20York&t=&z=13&ie=UTF8&iwloc=&output=embed">
        </iframe>
    </div>

</section>
```
