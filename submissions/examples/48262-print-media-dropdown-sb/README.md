# Print-Optimized Dropdown Component

Dropdown component with print media query optimization. Dropdowns show trigger button only when printed.

## Files

- `demo.html` - Dropdown demo with print optimization
- `style.css` - Dropdown styling with @media print rules

## Usage

Open `demo.html` in a browser to see the dropdown. Use Ctrl+P (Cmd+P on Mac) to see simplified print styling.

## What This Does

Adds print media query optimization to dropdown components:
- Hides dropdown menu when printing
- Shows trigger button as bordered element

## Why Useful

Dropdown menus are interactive elements that don't need to appear on printed pages. This optimization ensures clean print output.
