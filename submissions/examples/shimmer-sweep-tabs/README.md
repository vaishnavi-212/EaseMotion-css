# CSS Shimmer Sweep Tabs

Issue: #54463

## Description

A responsive tab navigation featuring a shimmer sweep animation on hover using only HTML and CSS.

## Features

- Pure HTML & CSS
- Smooth shimmer sweep effect
- Responsive layout
- No JavaScript
- Accessible with prefers-reduced-motion support

## Usage

```html
<div class="tabs">
    <button class="tab active">Home</button>
    <button class="tab">Projects</button>
    <button class="tab">Skills</button>
    <button class="tab">Contact</button>
</div>
```

Include:

```html
<link rel="stylesheet" href="style.css">
```

## Customization

Modify these values in `style.css`:

- Background color
- Hover color
- Animation duration
- Border radius
- Font size

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
