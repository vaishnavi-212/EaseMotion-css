# Animated Progress Bar Component

A clean progress bar component with animated fill and staggered animation delays.

## Files

- `demo.html` - Progress bar examples
- `style.css` - Full styling with animations

## Features

- Animated fill on load
- Staggered animation delays
- Gradient fill
- Rounded track and bar

## Design Philosophy

Minimal, informative progress bars with smooth animations that clearly show completion status.
