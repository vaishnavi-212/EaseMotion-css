# Ease Team Speedometer

## Overview

Ease Team Speedometer is a pure CSS animated performance gauge component.
It visually represents team progress, efficiency, or performance levels using
a smooth animated needle indicator.

## Features

- Pure CSS animation
- Responsive design
- No JavaScript required
- Easy to customize
- Smooth needle movement effect
