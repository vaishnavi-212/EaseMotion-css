# Ease System Health Dashboard

A responsive dashboard component that displays system resource usage using animated progress bars.

## Features

- CPU usage indicator
- RAM usage indicator
- Storage usage indicator
- Network usage indicator
- Smooth loading animation
- Responsive design
- Pure HTML & CSS
- No JavaScript required

## Folder Structure

```
ease-system-health-dashboard/
├── demo.html
├── style.css
└── README.md
```

## Usage

```html
<link rel="stylesheet" href="style.css">
```

Open `demo.html` in any browser.

## Customization

- Edit usage percentages.
- Change progress bar colors.
- Add more system metrics.
- Modify card size and spacing.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Fits EaseMotion CSS

This component uses semantic class names, lightweight CSS, responsive layout, and subtle animations while requiring no JavaScript.
