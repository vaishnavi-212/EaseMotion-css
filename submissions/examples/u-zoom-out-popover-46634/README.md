# CSS Zoom-Out Popover for Creative Portfolio Layouts

A **pure CSS Zoom-Out Popover** featuring a smooth zoom-out interaction designed for **Creative Portfolio** interfaces. The component is fully responsive, keyboard accessible, and built entirely with HTML and CSS.

## Features

- Pure HTML & CSS
- Smooth Zoom-Out animation
- Creative Portfolio design
- Fully responsive
- Keyboard accessible (`:focus-within`)
- CSS Custom Properties
- No JavaScript required
- Lightweight and reusable
