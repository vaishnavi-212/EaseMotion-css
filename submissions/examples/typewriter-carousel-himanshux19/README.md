# Typewriter Carousel

A responsive **Typewriter Carousel** component for the EaseMotion CSS library.

## Features

- Pure CSS
- No JavaScript
- Responsive layout
- Accessible HTML structure
- Typewriter heading animation
- Automatic carousel rotation
- Uses EaseMotion animation classes (`ease-fade-up`)

## Folder Structure

```
submissions/examples/typewriter-carousel-himanshux19/
│── demo.html
│── style.css
└── README.md
```

## Customization

Modify the following CSS variables:

```css
--ease-primary
--ease-bg
--ease-card
--ease-text
--ease-muted
--duration
```

## Accessibility

- Semantic headings
- High color contrast
- Responsive typography
- Keyboard-friendly content (no interaction required)

## Browser Support

Works in all modern browsers supporting CSS animations.
