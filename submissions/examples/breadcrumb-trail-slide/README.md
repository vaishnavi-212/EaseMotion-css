# Breadcrumb Trail Slide

This example adds a breadcrumb navigation trail where each crumb slides into place with a stagger.

## Files

- `demo.html` contains the breadcrumb navigation markup.
- `style.css` defines the staggered slide, focus state, and reduced-motion fallback.

## Highlights

- Uses semantic `nav`, ordered list, and `aria-current`.
- Keeps crumb height stable while animating with transforms.
- Includes visible keyboard focus styling for links.
- Shows static breadcrumbs for reduced-motion users.
