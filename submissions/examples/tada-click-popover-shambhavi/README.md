# Tada Click Popover – Glassmorphism UI

A standalone Glassmorphism Popover demonstrating a smooth Tada-Click interaction using pure HTML and CSS.

## Features

- Pure HTML & CSS
- Tada-style click animation
- Glassmorphism design
- Responsive layout
- Keyboard accessible using `:focus-within`
- CSS Custom Properties
- Zero JavaScript

## Files

```
demo.html
style.css
README.md
```

## Usage

Open `demo.html` directly in any modern browser.

Customize animation through CSS variables:

```css
--duration
--radius
--glass-bg
--popover-bg
```
