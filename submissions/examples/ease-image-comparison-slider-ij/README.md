# Image Comparison Slider

Before/after image comparison with draggable slider handle. Clip-paths adjust based on slider position via --ics-pos.

## Features

- Before/after image comparison
- Draggable slider handle
- Clip based on slider position via --ics-pos
- Smooth drag interaction

## Usage

Set `--ics-pos` (5-95) on `.ics-compare`. JS handles drag to update position.
