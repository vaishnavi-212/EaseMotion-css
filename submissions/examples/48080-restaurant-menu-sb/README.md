# Restaurant Menu Card Component

A beautifully designed restaurant menu card component with food item displays, pricing, and dietary badges. Perfect for food delivery apps, restaurant websites, and online ordering systems.

## Files

- `demo.html` - Menu grid with food items and cards
- `style.css` - Full styling with animations and responsive design

## Usage

Open `demo.html` in any modern browser to view the restaurant menu cards.

### HTML Classes

| Class | Description |
|-------|-------------|
| `.menu-container` | Main container for menu section |
| `.menu-grid` | Grid layout for menu cards |
| `.menu-card` | Individual food item card |
| `.card-badge` | Dietary tags (Chef's Pick, Vegan, Spicy) |
| `.card-image` | Food photo/icon area |
| `.card-content` | Text content area |
| `.dish-name` | Name of the dish |
| `.dish-description` | Dish description text |
| `.price` | Price display |
| `.add-btn` | Add to cart button |

## Features

- **Pure CSS implementation** - No JavaScript required
- **Dietary badges** - Visual tags for dietary preferences
- **Animated cards** - Staggered entrance animations
- **Floating food icons** - Subtle hover animation
- **Responsive grid** - Adapts to any screen size
- **Add button** - Interactive cart button with hover effects

## Design Philosophy

This component combines appetizing visual presentation with clear information hierarchy. The warm color palette and elegant typography create a premium dining experience, while the floating animations add subtle delight without compromising usability.
