# Shimmer Sweep Tooltip - E-Commerce Checkout Layouts

A CSS tooltip component with premium shimmer sweep animation designed for e-commerce checkout pages.

## Files

- `demo.html` - Demo page showing checkout UI with tooltips
- `style.css` - Tooltip styles and animations

## Usage

Open `demo.html` in a browser and hover over interactive elements to see the shimmer sweep tooltip animation.

## Features

### Shimmer Sweep Animation
- Premium green gradient sweep effect
- Bouncy scale entrance animation
- Smooth infinite loop

### E-Commerce Components
- Quantity adjustment buttons with tooltips
- Promo code input with apply button
- Primary checkout CTA button
- Trust badges with tooltip descriptions

### Design System
- Clean, professional checkout aesthetic
- Green accent color for trust and action
- Subtle shadows and borders

## Classes

| Class | Description |
|-------|-------------|
| `ease-shimmer-tooltip-ec-sb` | E-commerce shimmer tooltip |
| `ease-qty-btn-sb` | Quantity adjustment button |
| `ease-promo-input-sb` | Promo code input field |
| `ease-apply-btn-sb` | Apply promo button |
| `ease-checkout-btn-sb` | Primary checkout CTA |

## Usage

Add `data-tooltip="Your text"` attribute to any element with `ease-shimmer-tooltip-ec-sb` class:

```html
<button class="ease-shimmer-tooltip-ec-sb" data-tooltip="Secure checkout">
  Checkout
</button>
```
