# Print-Optimized Menu Component

Navigation menu with print media query optimization. Menus simplify when printed, hiding dropdowns.

## Files

- `demo.html` - Menu demo with print optimization
- `style.css` - Menu styling with @media print rules

## Usage

Open `demo.html` in a browser to see the navigation menu. Use Ctrl+P (Cmd+P on Mac) to see simplified print styling.

## What This Does

Adds print media query optimization to menu components:
- Removes dark background styling
- Shows links as simple text
- Hides dropdown menus

## Why Useful

Menus with dark backgrounds and dropdowns don't print well. This optimization ensures menus render as clean navigation lists.
