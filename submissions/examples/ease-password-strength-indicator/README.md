# Ease Password Strength Indicator

A reusable Password Strength Indicator component built with pure HTML and CSS.

## Features

- Animated strength bar
- Modern card UI
- Responsive design
- Gradient progress indicator
- Easy customization
- Pure HTML & CSS
- No JavaScript required

## Folder Structure

```
ease-password-strength-indicator/
├── demo.html
├── style.css
└── README.md
```

## Usage

Include the stylesheet:

```html
<link rel="stylesheet" href="style.css">
```

Open `demo.html` directly in your browser.

## Customization

- Change colors in `.strength-fill`
- Modify animation duration
- Customize labels
- Adjust card dimensions

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
