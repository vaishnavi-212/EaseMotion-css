# Timeline Step Pulse

This example adds a vertical timeline where the active step uses a subtle pulse to show progress.

## Files

- `demo.html` contains the ordered timeline markup.
- `style.css` defines the connector line, active pulse, and reduced-motion fallback.

## Highlights

- Uses an ordered list for timeline semantics.
- Keeps each step height stable across states.
- Shows completed, active, and pending states.
- Removes the active pulse when reduced motion is requested.
