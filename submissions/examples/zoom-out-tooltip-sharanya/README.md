# Zoom-Out Tooltip for Product Catalog Layout

A product catalog card with a CSS-powered zoom-out tooltip that appears when users interact with the product action button.

## Features

- Pure HTML and CSS
- Zoom-out tooltip animation
- Product card layout
- Easy to customize
- Responsive-friendly structure

## Files

- demo.html – Demonstrates the product card and tooltip.
- style.css – Contains all styling and animation.
- README.md – Component documentation.

## How to Use

1. Open `demo.html` in any modern browser.
2. Hover over the product action button to view the tooltip animation.

## Customization

You can modify:
- Colors
- Product title
- Price
- Tooltip text
- Animation duration

## Browser Support

- Chrome
- Edge
- Firefox
- Safari
