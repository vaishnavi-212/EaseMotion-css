# Ease Media Player Controls

A responsive media-player controls component with animated progress, equalizer bars, volume control, and accessible buttons.

## How to use

Open `demo.html` directly in a browser.

```html
<div class="control-row" aria-label="Media controls">
  <button class="control-btn" aria-label="Previous track">⏮</button>
  <button class="control-btn play-btn" aria-label="Play track">▶</button>
  <button class="control-btn" aria-label="Next track">⏭</button>
</div>
```

## Why it is useful

Media controls are common in music cards, podcasts, learning previews, video sections, and creator dashboards. This demo gives EaseMotion a polished, zero-dependency animated control pattern.

Features
Animated equalizer bars
Animated progress indicator
Responsive media player card
Accessible button labels
Keyboard focus states
Reduced-motion support
No external dependencies
