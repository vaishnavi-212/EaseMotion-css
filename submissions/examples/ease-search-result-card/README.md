# ease-search-result-card

A responsive search result card component with title, URL, description, tags, and smooth hover animation.

## Features

- Responsive layout
- Search result title & URL
- Description section
- Tag badges
- Hover elevation effect
- Pure HTML & CSS
- No JavaScript required

## Folder Structure

```
ease-search-result-card/
├── demo.html
├── style.css
└── README.md
```

## Usage

```html
<link rel="stylesheet" href="style.css">
```

Open `demo.html` directly in any browser.

## Customization

- Modify colors
- Add more tags
- Change typography
- Adjust shadows and animations

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
