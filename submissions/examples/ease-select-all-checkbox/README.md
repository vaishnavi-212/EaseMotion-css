# Ease Select All Checkbox

A modern, responsive **Select All Checkbox** component built using **HTML, CSS, and JavaScript**. It provides an intuitive way to select or deselect multiple options while automatically handling the indeterminate state when only some items are selected.

## ✨ Features

- Select or deselect all checkboxes with a single click
- Automatic indeterminate state for partial selections
- Live selected items counter
- Animated progress bar
- Smooth hover and selection animations
- Responsive and modern UI
- Easy to customize and integrate

## 🚀 How It Works

- Clicking **Select All** selects every option.
- Unchecking **Select All** deselects all options.
- Selecting individual items automatically updates the **Select All** checkbox.
- The progress bar and selected counter update in real time.
- When all items are selected, the component displays a subtle celebration animation.
