# Print-Optimized Code Block Component

Code block styling with print media query optimization. Code blocks render with white backgrounds when printed.

## Files

- `demo.html` - Code block demo with print optimization
- `style.css` - Code block styling with @media print rules

## Usage

Open `demo.html` in a browser to see the code block styling. Use Ctrl+P (Cmd+P on Mac) to see simplified print styling.

## What This Does

Adds print media query optimization to code block components:
- Converts dark theme to white background
- Maintains readability with black text

## Why Useful

Dark-themed code blocks with colored syntax are not suitable for printing. This optimization ensures code renders with clean, readable styling on paper.
