# Ease Pricing Plan Card

A responsive CSS-only pricing plan card component featuring multiple pricing tiers, a highlighted featured plan, and smooth hover animations.

## Features

- 💳 Three pricing plans
- ⭐ Featured plan highlight
- 🚀 Hover animation
- 📱 Responsive layout
- 🎨 Pure HTML & CSS
- ♻ Easy to customize

## Folder Structure

```
ease-pricing-plan-card/
├── demo.html
├── style.css
└── README.md
```

## Usage

```html
<link rel="stylesheet" href="style.css">

<div class="card featured">
    <div class="badge">POPULAR</div>
    <h2>Pro</h2>
    <h1>$29<span>/mo</span></h1>
</div>
```

Use the `featured` class to highlight the recommended pricing plan.

Open `demo.html` directly in any browser.

- Chrome

- Safari
