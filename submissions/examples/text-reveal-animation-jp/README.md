# Text Reveal Animation (`ease-text-reveal`)

A sleek entrance animation where text appears to slide up smoothly from behind an invisible mask. This creates a highly professional, modern typographic entrance effect that is extremely popular on agency websites, portfolios, and landing pages.

**What does this do?**
Provides utility classes `.ease-text-reveal-wrapper` and `.ease-text-reveal` to create a smooth, masked vertical slide-up animation for text elements.

**How is it used?**
```html
<h1 class="ease-text-reveal-wrapper">
  <span class="ease-text-reveal">Welcome to EaseMotion</span>
</h1>
```
*Note: Ensure the animating text is wrapped in an element with `.ease-text-reveal-wrapper` to mask the animation before it enters the viewport.*

**Why is it useful?**
- **Premium Aesthetics**: It provides a highly requested, modern animation technique.
- **Animation-first**: It's a pure CSS keyframe animation requiring no JavaScript.
- **Human-readable**: `ease-text-reveal` is intuitive and clearly describes the visual outcome.
