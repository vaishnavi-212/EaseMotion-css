# Ease Teddy Bear Counter

A cute animated teddy bear counter component built using pure HTML and CSS.

## ✨ Features

- Teddy bear bounce animation
- Modern card UI
- Smooth hover effects
- Responsive design
- CSS custom properties
- No JavaScript required

## 📌 Usage

Add the counter structure:

```html
<div class="counter-card">

  <div class="teddy">🧸</div>

  <div class="count">
    12
  </div>

</div>
