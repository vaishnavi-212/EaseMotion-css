# Skew Active Popover – Product Catalog

A responsive Product Catalog Popover featuring a Skew Active animation built entirely with HTML and CSS.

## Features

- Pure HTML & CSS
- Skew Active animation
- Product catalog UI
- Responsive layout
- Keyboard accessible
- CSS custom properties
- Zero JavaScript

## Files

```
demo.html
style.css
README.md
```

## Usage

Open `demo.html` directly in your browser.

Customize:

```css
--primary
--duration
--radius
--skew
```
