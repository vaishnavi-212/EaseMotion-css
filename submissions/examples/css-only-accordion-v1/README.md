# CSS-only Accordion Component

A fully functional, animated accordion component that relies entirely on CSS, using native `<details>` and `<summary>` elements. It features smooth expansion/collapse animations, a rotating chevron icon, and hover interactions aligned with the framework's "animation-first" philosophy.

## Files
- `demo.html` - Interactive demonstration of the accordion.
- `style.css` - The CSS implementation utilizing EaseMotion tokens.

## Details
This addresses the "CSS-only accordion" requirement currently listed as "Planned" in the project Roadmap. It requires zero JavaScript.
