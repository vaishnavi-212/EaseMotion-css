# Ease Microservice Dependency Map

## Overview
A CSS-only component for visualizing service dependencies.

## Features
- Connected service nodes
- Responsive layout
- Hover animation
- CSS variables
- Pure HTML & CSS

## Usage
Open `demo.html` in any browser.

## Browser Support
Chrome, Firefox, Edge, Safari.

## Why EaseMotion CSS?
A reusable animation-first component for architecture dashboards, DevOps tools, cloud platforms, and backend monitoring systems.
