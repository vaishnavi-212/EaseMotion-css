# List Item Reorder

A list row pattern that lifts and reveals a handle on hover or keyboard focus.

## Files

- `demo.html` - focusable task list rows.
- `style.css` - row lift, handle reveal, focus-visible state, and reduced-motion support.

## Highlights

- Keeps row height stable during interaction.
- Mirrors hover feedback on keyboard focus.
- Uses subtle handle reveal for reorder affordance.
