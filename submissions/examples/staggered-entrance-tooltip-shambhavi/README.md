# Staggered Entrance Tooltip – SaaS Showcase

A standalone responsive tooltip demonstrating a staggered entrance animation using pure HTML and CSS.

## Features

- Pure HTML & CSS
- Staggered entrance animation
- SaaS-inspired clean design
- Responsive layout
- Keyboard accessible
- Customizable CSS variables
- Zero JavaScript

## Files

```
demo.html
style.css
README.md
```

## Usage

Open `demo.html` directly in your browser.

Customize animation values using CSS variables:

```css
--duration
--delay
--distance
```
