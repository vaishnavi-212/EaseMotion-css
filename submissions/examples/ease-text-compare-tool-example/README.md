# Ease Text Compare Tool

## Overview

The **Ease Text Compare Tool** is a responsive example component that allows users to compare two blocks of text. It provides instant feedback indicating whether the texts match or differ while demonstrating clean UI design, responsive layouts, and smooth CSS transitions.

---

## Features

- 📝 Side-by-side text comparison
- 🔍 Compare button
- ✅ Match and difference status display
- 🎨 Responsive design
- ✨ Smooth hover and transition effects
- ⚙️ Customizable using CSS custom properties

---

## CSS Custom Properties

| Variable | Description |
|----------|-------------|
| `--primary` | Primary accent color |
| `--bg` | Background color |
| `--text` | Text color |
| `--duration` | Transition duration |

Example:

```css
:root {
    --primary: #2563eb;
    --bg: #ffffff;
    --text: #1f2937;
    --duration: 300ms;
}
```

## How to Use

1. Open `demo.html` in a modern web browser.
2. Enter text into both text areas.
3. Click **Compare Text**.
4. View whether the texts match or differ.
