# Magnetic Pull Tooltip – Minimalist Tech

A clean, reusable Magnetic Pull Tooltip built using pure CSS for minimalist technology interfaces.

## Features

- Pure CSS
- No JavaScript
- Smooth magnetic pull animation
- Responsive layout
- Keyboard accessible
- CSS Custom Properties for customization
- Lightweight and reusable
