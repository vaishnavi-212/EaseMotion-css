# Ease Chat Message Bubble

A responsive chat message bubble component built with pure HTML and CSS. It features animated incoming and outgoing messages with timestamps, making it ideal for messaging interfaces, chat applications, customer support dashboards, and collaboration tools.

## Features

- 💬 Incoming & outgoing chat bubbles
- ✨ Smooth entry animation
- 🕒 Message timestamps
- 📱 Responsive layout
- 🎨 Pure HTML & CSS
- ♻️ Easy to customize

## Folder Structure

```
ease-chat-message-bubble/
├── demo.html
├── style.css
└── README.md
```

## Usage

```html
<div class="message incoming">
    <p>Hello!</p>
    <span>10:15 AM</span>
</div>

<div class="message outgoing">
    <p>Hi there!</p>
    <span>10:16 AM</span>
</div>
```

Use:

- `.incoming` for received messages
- `.outgoing` for sent messages

Open `demo.html` directly in your browser.
