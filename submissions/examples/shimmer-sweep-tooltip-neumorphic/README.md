# Shimmer Sweep Tooltip - Neumorphic Soft Layouts

## Overview

A pure CSS animated tooltip featuring a smooth shimmer sweep effect designed for neumorphic soft UI interfaces.

The component creates a modern lighting sweep animation without JavaScript.

## Features

- Pure CSS animation
- Smooth shimmer sweep effect
- Neumorphic design style
- Responsive layout
- Keyboard accessible
- Custom CSS variables
- Zero JavaScript dependency

## Usage

```html
<div class="tooltip-wrapper">

<button class="tooltip-trigger">
 Hover Me
</button>

<div class="tooltip">
 Tooltip Content
</div>

</div>
