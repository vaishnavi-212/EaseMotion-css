# Ease Calendar Event Card

A modern CSS-only calendar event card for displaying event details with an elegant layout and smooth hover animation.

## Features

- 📅 Stylish calendar date section
- 🏷️ Status badge (Upcoming)
- ✨ Smooth hover animation
- 🎨 Pure HTML & CSS
- 📱 Responsive design
- 🚫 No JavaScript required

## Folder Structure

```
ease-calendar-event-card/
├── demo.html
├── style.css
└── README.md
```

## Usage

```html
<link rel="stylesheet" href="style.css">

<div class="event-card">
    <div class="event-date">
        <span class="day">25</span>
        <span class="month">JUL</span>
    </div>

    <div class="event-info">
        <span class="status upcoming">Upcoming</span>
        <h2>Team Sprint Planning</h2>
        <p>🕒 10:00 AM - 11:30 AM</p>
        <p>📍 Conference Room A</p>
    </div>
</div>
```

Open `demo.html` directly in your browser.
