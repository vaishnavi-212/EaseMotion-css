# Scale-Hover Carousel – Interactive Interface

A pure CSS Scale-Hover Carousel component for EaseMotion CSS.

## Features

- Pure CSS
- Responsive grid carousel
- Smooth scale-hover animation
- Keyboard accessible (`tabindex` + `:focus-visible`)
- Uses CSS custom properties
- Interactive Interface inspired design

## Custom Properties

```css
--ease-duration
--ease-scale
--ease-easing
--primary
--secondary
```
