# Wave Color Picker 🌊🎨

A smooth and modern **Wave Color Picker** component built with pure CSS.  
This component provides an interactive product color selection experience with animated liquid waves and responsive design.

## ✨ Features

- Pure CSS color picker (No JavaScript required)
- Smooth liquid wave animation
- Multiple color themes:
  - Purple
  - Ocean
  - Sunset
  - Mint
  - Rose
- Dynamic background and wave color changes
- Glassmorphism card design
- Responsive layout for mobile and desktop
- Accessible radio-button based controls
- Uses EaseMotion style variables and animations
