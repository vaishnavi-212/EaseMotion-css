# Zoom-In Carousel – Interactive Interface

A pure CSS Zoom-In Carousel component built for EaseMotion CSS.

## Features

- Pure CSS (No JavaScript)
- Responsive design
- Keyboard accessible
- Zoom-In transition between slides
- Uses CSS custom properties for easy customization
- Interactive interface themed layout

## Custom Properties

```css
--ease-duration
--ease-scale
--ease-easing
--primary
--secondary
```
