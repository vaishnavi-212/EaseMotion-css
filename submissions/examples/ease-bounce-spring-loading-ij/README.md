# Bounce Spring Loading

Loading dots bounce with spring-like motion. Each dot has staggered bounce animation.

## Features

- Spring-like bounce animation
- Staggered delays via `--bsl-i`
- Customizable primary color
- Pure CSS, no JS required

## Usage

Add `.bsl-dot` elements inside `.bsl-container`. Set `--bsl-i` (0-based index) for stagger delay.
