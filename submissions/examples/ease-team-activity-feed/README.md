# Ease Team Activity Feed

A lightweight CSS-only activity feed component for displaying team updates,
events, and notifications with smooth animations.

## Features

- Pure HTML and CSS
- Smooth hover interaction
- Responsive design
- Clean timeline-style layout
- No JavaScript required

## Usage

```html
<div class="activity">
  <div class="icon">🚀</div>
  <div>
    <h3>Feature deployed</h3>
    <p>New update released</p>
  </div>
</div>
