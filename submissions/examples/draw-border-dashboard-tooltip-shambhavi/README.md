# Draw Border Line Tooltip – Dashboard Analytics

A responsive Dashboard Analytics tooltip featuring a Draw Border Line interaction using only HTML and CSS.

## Features

- Pure HTML & CSS
- Draw Border Line animation
- Dashboard Analytics inspired design
- Responsive layout
- Keyboard accessible
- CSS custom properties
- Zero JavaScript

## Files

```
demo.html
style.css
README.md
```

## Usage

Open `demo.html` directly in your browser.

Customize using:

```css
--primary
--duration
--radius
--border-color
```
