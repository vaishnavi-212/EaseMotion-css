# Magnetic Hover Effect

**What does this do?**
Creates a "magnetic" pull effect where the element smoothly pops out and floats towards the user on hover, using a playful cubic-bezier transition.

**How is it used?**
Apply the class to a button or card:
```html
<button class="magnetic-button">Hover Me</button>
```

**Why is it useful?**
It adds a lively, interactive, and tactile feel to call-to-action buttons, fitting perfectly into the EaseMotion CSS philosophy of human-readable, motion-first design.
