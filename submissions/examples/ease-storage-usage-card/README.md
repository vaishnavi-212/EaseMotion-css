# Ease Storage Usage Card

A modern and responsive Storage Usage Card built using pure HTML and CSS.

## Features

- Animated storage progress bar
- Storage usage statistics
- Responsive card layout
- Modern UI
- CSS-only animation
- Easy customization
- No JavaScript required

## Folder Structure

```
ease-storage-usage-card/
├── demo.html
├── style.css
└── README.md
```

## Usage

```html
<link rel="stylesheet" href="style.css">
```

Open `demo.html` directly in any browser.

## Customization

- Change storage values
- Modify progress bar colors
- Update animation duration
- Customize card size and typography

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
