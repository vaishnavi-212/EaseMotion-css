ease-navbar — @media print Fix

The ease-navbar component uses semi-transparent backgrounds, backdrop-filters, transitions, and box-shadows. When printed, these result in bad contrast, ink waste, and overlapping text on multi-page documents. Adding a @media print block optimizes the navbar for printing.

Press Ctrl + P (or Cmd + P on macOS) to see the print layout.
