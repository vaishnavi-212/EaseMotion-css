# Blur Entrance Popover – Modern SaaS

A responsive Modern SaaS popover featuring a smooth Blur Entrance animation built entirely with HTML and CSS.

## Features

- Pure HTML & CSS
- Blur Entrance animation
- Modern SaaS dashboard design
- Responsive layout
- Keyboard accessible
- CSS custom properties
- Zero JavaScript
