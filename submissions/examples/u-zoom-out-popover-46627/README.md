# CSS Zoom-Out Popover for SaaS Showcase Layouts

A **pure CSS Zoom-Out Popover** designed for **SaaS Showcase** interfaces. The component uses a smooth zoom-out entrance animation, modern SaaS styling, responsive layout, keyboard accessibility, and CSS custom properties without JavaScript.

## Features

- Pure HTML & CSS
- Smooth Zoom-Out animation
- SaaS Showcase design
- Fully responsive
- Keyboard accessible (`:focus-within`)
- CSS Custom Properties
- Lightweight
- No JavaScript
