# CSS Skew Active Carousel

A modern portfolio-style carousel component built using only HTML and CSS.

## Features

- Pure HTML & CSS
- Smooth skew-active transition
- Responsive layout
- Keyboard accessible (`:focus-visible`)
- CSS custom properties for easy customization
- Modern portfolio-inspired design
- No JavaScript required

## Folder Structure

```
examples/
└── skew-active-carousel/
    ├── demo.html
    ├── style.css
    └── README.md
```

## Customization

Modify the CSS variables:

```css
:root{
  --active:#3b82f6;
  --skew:10deg;
  --duration:.45s;
  --active-scale:1.08;
}
```
