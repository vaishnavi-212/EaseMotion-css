# Print-Optimized Notification Component

Notification/toast components with print media query optimization. Notifications render with simplified border styling when printed.

## Files

- `demo.html` - Notification demo with print optimization
- `style.css` - Notification styling with @media print rules

## Usage

Open `demo.html` in a browser to see the styled notifications. Use Ctrl+P (Cmd+P on Mac) to see simplified print styling.

## What This Does

Adds print media query optimization to notification components, removing backgrounds and shadows while keeping border styling for readability on paper.

## Why Useful

Notification backgrounds waste ink when printed. This optimization ensures notifications remain readable with border styling while saving ink.
