# Elastic Slide Dropdown

A modern dropdown component built using pure HTML5 and CSS3 with an elastic slide animation for creative portfolio layouts.

## Features

- Pure HTML & CSS
- Elastic slide dropdown animation
- Hover and keyboard (`:focus-within`) support
- Responsive design
- Modern UI
- No JavaScript required
- Supports `prefers-reduced-motion`

## Folder Structure

```
elastic-slide-dropdown/
├── demo.html
├── style.css
└── README.md
```

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Usage

Open `demo.html` in any modern browser to view the component.
