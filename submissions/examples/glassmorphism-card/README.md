# Glassmorphism Card

Frosted glass effect cards using `backdrop-filter`. Includes gradient borders, inner glow, and floating animation variants.

## Demo

Open `demo.html` in any modern browser.

## Features

- ✅ **Pure CSS** — zero JavaScript
- ✅ **Frosted glass** — `backdrop-filter: blur()` effect
- ✅ **Gradient border** — animated rainbow border
- ✅ **Inner glow** — subtle white highlight
- ✅ **Floating animation** — gentle hover animation
- ✅ **Dark variant** — darker glass for contrast
- ✅ **Responsive** — adapts to mobile
- ✅ **Accessible** — respects `prefers-reduced-motion`

## Usage

### Basic Glass Card

```html
&lt;div class="glass"&gt;
  &lt;h2&gt;Title&lt;/h2&gt;
  &lt;p&gt;Content goes here...&lt;/p&gt;
&lt;/div&gt;
