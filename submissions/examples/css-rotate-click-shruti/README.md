# Rotate Click Popover – Accessible Web

A pure CSS Rotate Click Popover designed for accessible web interfaces.

## Features

- Pure CSS
- No JavaScript
- Responsive
- Keyboard accessible
- Uses CSS Custom Properties
- Smooth rotate-click animation
- Lightweight and reusable
