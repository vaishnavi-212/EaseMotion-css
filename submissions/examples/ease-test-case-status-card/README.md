# Ease Test Case Status Card

## Overview

A reusable test case status card component for displaying software testing information such as test name, status, execution details, and environment.

Built using pure CSS with smooth animation and responsive design.

## Features

- Clean test case UI card
- Status badge support
- Smooth entrance animation
- Responsive layout
- Hover interaction
- CSS variable customization
- No JavaScript required

## Usage

```html
<div class="test-card">

<h2>Login Test Case</h2>

<span class="status success">
Passed
</span>

<p>
Test completed successfully.
</p>

</div>
