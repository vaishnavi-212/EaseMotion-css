# Optimized Pulse Animation

A responsive Pulse Animation example demonstrating CSS animation performance best practices using GPU-friendly properties.

## Features

- Pure HTML & CSS
- Optimized Pulse animation
- GPU accelerated (transform + opacity)
- Responsive layout
- Semantic HTML
- Reusable components
- Zero JavaScript

## Performance Improvements

- Animates only transform and opacity
- Avoids layout-triggering properties
- Uses will-change for smoother rendering
- Lightweight reusable animation
- Works efficiently across modern browsers

## Files

```
demo.html
style.css
README.md
```

## Usage

Open `demo.html` directly in your browser.

Customize animation speed:

```css
--duration:1.4s;
```

Customize color:

```css
--primary:#4f46e5;
```
