# Print-Optimized Badge Component

A collection of badge/label components with print media query optimization. Badges render as simple bordered elements when printed.

## Files

- `demo.html` - Badge demos with print optimization
- `style.css` - Badge styling with @media print rules

## Usage

Open `demo.html` in a browser to see the styled badges. Use Ctrl+P (Cmd+P on Mac) to see simplified print styling.

## What This Does

Adds print media query optimization to badge components, removing backgrounds and rendering badges as simple bordered elements with black text.

## Why Useful

Badges with colorful backgrounds waste ink when printed. This optimization ensures badges render as simple bordered labels on paper.
