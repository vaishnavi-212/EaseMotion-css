# 3D Flip Carousel – Glassmorphism UI

A pure CSS 3D Flip Carousel component built for EaseMotion CSS.

## Features

- Pure CSS (No JavaScript)
- Glassmorphism-inspired UI
- Smooth 3D Flip transition
- Responsive layout
- Keyboard accessible
- Customizable using CSS custom properties

## Custom Properties

```css
--ease-duration
--ease-easing
--ease-rotate
--glass-bg
--glass-border
--accent
```
