# EaseMotion CSS Dashboard Showcase

## What does this do?

A comprehensive dashboard layout demo showcasing 17 EaseMotion CSS components working together in a realistic admin interface — including sidebar navigation, stats cards, tabs, progress bars, pagination, badges, chips, tags, avatars, toast notifications, modal dialogs, skeleton loaders, and theme toggling.

## How is it used?

Open `demo.html` in a browser. The dashboard loads with:

- **Sidebar** — collapsible navigation with badge counters and active state
- **Stats row** — four interactive stat cards with hover animations
- **Tabs** — Overview, Projects, Team, and Tasks panels
- **Progress bars** — four task progress indicators with color variants
- **Transaction table** — paginated data table with status badges
- **Skeleton loader** — toggleable loading placeholders
- **Toast notifications** — auto-dismissing success messages
- **Modal dialog** — opens on stat card click with confirm/cancel
- **Dark mode** — toggle button to switch between light and dark themes

## Why is it useful?

This showcase demonstrates real-world composition of EaseMotion CSS components. It shows how `ease-card`, `ease-badge`, `ease-chip`, `ease-tag`, `ease-tabs`, `ease-progress`, `ease-skeleton`, `ease-toast`, `ease-modal-overlay`, `ease-modal`, `ease-avatar`, `ease-pagination`, `ease-sidebar`, `ease-navbar-glass`, and utility classes can be combined into a polished production dashboard. It follows EaseMotion's zero-dependency philosophy — no JavaScript frameworks, no build tools, just HTML + CSS + vanilla JS.
