# Ease Environment Variable Panel

## Overview
A CSS-only component for displaying environment variables with visibility badges.

## Features
- Public and private labels
- Responsive layout
- Hover animation
- CSS variables
- Pure HTML & CSS

## Usage
Open `demo.html` directly in any browser.

## Browser Support
Chrome, Firefox, Edge, Safari.

## Why EaseMotion CSS?
A reusable animation-first component for DevOps dashboards, CI/CD pipelines, cloud platforms, and developer portals.
