# Print-Optimized Drawer Component

Drawer/sidebar component with print media query optimization. Drawers are hidden when printing.

## Files

- `demo.html` - Drawer demo with print optimization
- `style.css` - Drawer styling with @media print rules

## Usage

Open `demo.html` in a browser to see the drawer. Use Ctrl+P (Cmd+P on Mac) to see drawer hidden in print mode.

## What This Does

Adds print media query optimization to drawer components, hiding them when printing. Drawers are navigation elements that don't need to appear on printed pages.

## Why Useful

Drawers are typically used for mobile navigation or quick access panels. They have no purpose in printed documents.
