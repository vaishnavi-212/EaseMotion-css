# Toast Progress Sweep

This example adds a status toast with a progress sweep along the bottom edge.

## Files

- `demo.html` contains the toast markup.
- `style.css` defines the entry motion, progress sweep, and reduced-motion fallback.

## Highlights

- Uses `role="status"` for the demo notification.
- Keeps the toast height stable while progress animates.
- Separates entry motion from the progress indicator.
- Shows a complete progress bar when reduced motion is requested.
