# Animated Gradient Border Card

A modern, highly requested UI component featuring a smooth, continuously spinning gradient border. Perfect for pricing tables, premium feature callouts, or highlighting important user content.

## Features
- **CSS Houdini `@property`**: Uses modern CSS variables with the `@property` rule to smoothly animate a `conic-gradient` angle.
- **Sleek Masking**: The inner content block masks the background gradient to create a perfect, even border glow.
- **Interactive**: Includes a subtle hover lift effect to make the card feel tactile.

## How to use

```html
<div class="ease-gradient-border-card">
    <div class="ease-gradient-content">
        <h2>Pro Plan</h2>
        <p>Unlock all premium features and components for your next big project.</p>
        <button class="ease-gradient-btn">Get Started</button>
    </div>
</div>
```
