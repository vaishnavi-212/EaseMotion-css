# Rating Star Spark

This example adds a compact rating control with a star spark effect on hover and focus.

## Files

- `demo.html` contains the rating card and star buttons.
- `style.css` defines the spark animation, focus state, and reduced-motion fallback.

## Highlights

- Uses real buttons for rating interactions.
- Keeps each star at a stable square size.
- Applies the same visual treatment to hover and keyboard focus.
- Removes transform and spark animation when reduced motion is requested.
