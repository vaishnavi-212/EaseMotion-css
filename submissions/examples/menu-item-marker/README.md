# Menu Item Marker

This example adds a vertical menu where the active marker slides into place.

## Files

- `demo.html` contains the navigation menu markup.
- `style.css` defines the active marker, hover and focus states, and reduced-motion fallback.

## Highlights

- Uses semantic navigation links.
- Keeps link height stable while the marker animates.
- Applies the same marker treatment to hover and keyboard focus.
- Removes marker transition timing for reduced-motion users.
