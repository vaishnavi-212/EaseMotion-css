# Print-Optimized Card Component

Card components with print media query optimization. Cards render with simplified styling and no shadows when printed.

## Files

- `demo.html` - Card demos with print optimization
- `style.css` - Card styling with @media print rules

## Usage

Open `demo.html` in a browser to see the styled cards. Use Ctrl+P (Cmd+P on Mac) to see simplified print styling.

## What This Does

Adds print media query optimization to card components:
- Removes shadows in print mode
- Adds borders for definition
- Hides decorative images
- Simplifies buttons and badges

## Why Useful

Cards with shadows and images waste ink when printed. This optimization ensures cards render as clean text-based content blocks on paper.
