# CSS Blur Entrance Carousel

A responsive Blur-Entrance Carousel built using only HTML and CSS.

## Features

- Pure HTML & CSS
- Blur entrance animation
- Responsive layout
- Keyboard accessible
- CSS Custom Properties
- Modern SaaS design
- No JavaScript

## Folder Structure

```text
examples/
└── blur-entrance-carousel-saas/
    ├── demo.html
    ├── style.css
    └── README.md
```

## Customization

Modify the CSS variables:

```css
:root{
  --primary:#2563eb;
  --blur:14px;
  --duration:.7s;
  --radius:18px;
}
```
