# Print-Optimized Icon Component

Icon components with print media query optimization. Icons are hidden when printing.

## Files

- `demo.html` - Icon demo with print optimization
- `style.css` - Icon styling with @media print rules

## Usage

Open `demo.html` in a browser to see the styled icons. Use Ctrl+P (Cmd+P on Mac) to see icons hidden in print mode.

## What This Does

Adds print media query optimization to icon components, hiding them when printing. Icons are decorative elements that don't need to appear on printed pages.

## Why Useful

Icons with colored backgrounds and gradients waste ink and space when printed. Hiding them produces cleaner, more professional output.
