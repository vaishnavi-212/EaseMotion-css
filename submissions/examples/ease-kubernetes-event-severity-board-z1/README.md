# Ease Kubernetes Event Severity Board

## Overview

A responsive HTML and CSS dashboard component for visualizing Kubernetes events based on severity levels.

## Features

- Event severity overview
- Active event monitoring
- Warning event visibility
- Critical event alerts
- Responsive design
- Pure HTML & CSS
- CSS variables
- Hover effects and animations

## Folder Structure

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

Provides a reusable observability component for monitoring Kubernetes event severity, improving operational awareness, and enabling faster incident response.
