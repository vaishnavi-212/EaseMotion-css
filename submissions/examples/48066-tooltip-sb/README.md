# CSS Tooltip Component

A pure CSS tooltip component with four positioning options (top, bottom, left, right) and smooth animations.

## Files

- `demo.html` - Tooltip examples
- `style.css` - Full styling

## Features

- Pure CSS implementation
- Four positioning options
- Smooth fade animations
- Accessible with aria-labels

## Design Philosophy

Clean, minimal tooltips that appear on hover with directional arrows and smooth transitions.
