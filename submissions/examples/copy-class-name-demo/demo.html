Copy Class Name Button for Demo Cards

A self-contained pattern for adding a one-click "copy class name" button to animation demo cards, so developers can grab a class name without manually selecting text. Click the 📋 icon on any card below to try it.

How it works

  • Each demo card renders its EaseMotion class name in a <code> tag next to a small copy button.
  • Clicking the button reads the class name from a data-class attribute and writes it to the clipboard via the Clipboard API.
  • A short toast confirms the copy ("Class name copied!") and the button icon briefly swaps to a checkmark for visual feedback.
  • No dependencies — plain HTML, CSS, and vanilla JS, matching EaseMotion CSS's zero-dependency philosophy.
