# Copy Class Name Demo

## Description
A self-contained example demonstrating a "Copy Class Name" button for animation cards.

## Features
- Copies the animation class name to the clipboard
- Displays a toast notification after copying
- Shows temporary checkmark feedback
- No external dependencies

## Files
- demo.html
- style.css
- script.js

## Usage
Open `demo.html` in any modern browser and click the 📋 button to copy the animation class name.
