# Animated Timeline Progress Card

A responsive **CSS-only Animated Timeline Progress Card** that visually represents multi-step workflows such as onboarding, order tracking, project milestones, or progress indicators.

## Features

- ✅ Pure HTML & CSS
- ✅ Responsive layout
- ✅ Animated active step
- ✅ Hover lift effect
- ✅ Completed, active, and pending states
- ✅ No JavaScript required

## Folder Structure

```
animated-timeline-progress-card/
├── demo.html
├── style.css
└── README.md
```

## Usage

1. Open `demo.html` in any modern web browser.
2. All styling is contained in `style.css`.
3. Customize colors, text, or number of steps as needed.

## Customization

You can easily modify:

- Timeline colors
- Card background
- Number of steps
- Animation speed
- Hover effects
- Typography

## Browser Support

Works in all modern browsers including:

- Chrome
- Edge
- Firefox
- Safari
