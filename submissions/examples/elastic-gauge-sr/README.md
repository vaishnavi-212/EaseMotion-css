# Elastic Gauge

A reusable Elastic Gauge component inspired by neon design patterns for EaseMotion CSS.

## What it does

This component displays a gauge with an animated needle and a neon glow effect. It is built using only HTML and CSS, making it easy to use without any JavaScript.

## Features

- Pure HTML and CSS
- No JavaScript required
- Elastic needle animation
- Neon-inspired design
- Responsive layout
- Keyboard accessible
- Uses EaseMotion animation classes

## Folder Structure

```
elastic-gauge-sr/
├── demo.html
├── style.css
└── README.md
```

## Usage

Add the stylesheet to your page.

```html
<link rel="stylesheet" href="style.css">
```

Then open `demo.html` in a browser to see the component.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Notes

This component is intended as a reusable UI element for dashboards, analytics pages, monitoring interfaces, and similar layouts.
