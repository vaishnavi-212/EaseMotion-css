# Ease Serverless Function Invocations

## Overview
A pure HTML and CSS dashboard component for monitoring serverless function performance and execution status.

## Features
- Function execution cards
- Invocation statistics
- Average execution time
- Cold start information
- Deployment status badges
- Responsive layout
- CSS-only animations

## Folder Structure
demo.html
style.css
README.md

## Usage
Open `demo.html` directly in any modern browser.

## Browser Support
- Chrome
- Firefox
- Edge
- Safari

## Responsive
Optimized for desktop, tablet, and mobile devices.

## Why EaseMotion CSS?
Provides a reusable animation-first UI component for monitoring serverless workloads across AWS Lambda, Azure Functions, Google Cloud Functions, Cloudflare Workers, and similar platforms.
