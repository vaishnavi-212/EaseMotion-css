# Skew Active Popover – Interactive Interface

A responsive Interactive Interface Popover built with HTML and CSS featuring a smooth Skew Active animation.

## Features

- Pure HTML & CSS
- Skew Active animation
- Interactive dashboard layout
- Keyboard accessible
- Responsive
- CSS Custom Properties
- Zero JavaScript

## Files

```
demo.html
style.css
README.md
```

## Usage

Open `demo.html` directly in your browser.

Customize:

```css
--primary
--duration
--radius
--skew
```
