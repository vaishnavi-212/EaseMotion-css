# CSS Elastic Bounce Popover

A responsive **Elastic Bounce Popover** for **Product Catalog Layouts** built entirely with HTML and CSS.

## Features

- Pure HTML & CSS
- Elastic Bounce Animation
- Responsive Product Card
- Keyboard Accessible (`:focus-visible`)
- Hover & Focus Popover
- CSS Custom Properties
- No JavaScript

## Technologies

- HTML5
- CSS3

## CSS Variables

```css
--bounce-duration
--bounce-scale
--primary
--secondary
--popover-bg
--shadow
--transition
```
