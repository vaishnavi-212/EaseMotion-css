# Ease Reading Progress Bar

A modern CSS-only animated reading progress bar for the EaseMotion CSS library.

## Features

- 📖 Reading progress indicator
- ✨ Smooth animation
- 🎨 Pure HTML & CSS
- 📱 Responsive
- 🚫 No JavaScript required
- ⚡ Lightweight and reusable

## Folder Structure

```
ease-reading-progress-bar/
├── demo.html
├── style.css
└── README.md
```

## Usage

Include the stylesheet:

```html
<link rel="stylesheet" href="style.css">
```

Example:

```html
<div class="progress-container">
    <div class="progress-bar"></div>
</div>
```

Open `demo.html` directly in your browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
