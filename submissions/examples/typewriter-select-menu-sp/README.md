# Typewriter Select Menu

A Startup-inspired Select Menu with a typewriter heading animation built using pure HTML and CSS.

## Features

- ⌨️ Typewriter text animation
- 🎨 Startup-inspired minimal UI
- 📱 Responsive layout
- ♿ Keyboard accessible
- 🚫 Pure HTML & CSS (No JavaScript)

## File Structure

```
typewriter-select-menu-sp/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in your browser and interact with the animated select menu.
