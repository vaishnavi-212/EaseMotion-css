# Print-Optimized Breadcrumb Component

Breadcrumb component with print media query optimization. Breadcrumbs render with simplified styling when printed.

## Files

- `demo.html` - Breadcrumb demo with print optimization
- `style.css` - Breadcrumb styling with @media print rules

## Usage

Open `demo.html` in a browser to see the styled breadcrumbs. Use Ctrl+P (Cmd+P on Mac) to see simplified print styling.

## What This Does

Adds print media query optimization to breadcrumb components, removing backgrounds and simplifying styling for clean printed output.

## Why Useful

Breadcrumbs with colored backgrounds and decorative styling can look cluttered when printed. This optimization ensures breadcrumbs render as clean, readable navigation aids.
