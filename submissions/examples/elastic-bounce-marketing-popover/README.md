# CSS Elastic Bounce Popover

A responsive **Elastic Bounce Popover** designed for **Marketing Landing Page Layouts**, built entirely with HTML and CSS.

## Features

- Pure HTML & CSS
- Elastic Bounce Animation
- Marketing Landing Page Design
- Responsive Layout
- Keyboard Accessible (`:focus-visible`)
- Hover & Focus Popover
- CSS Custom Properties
- No JavaScript

## Technologies

- HTML5
- CSS3

## CSS Variables

```css
--primary
--secondary
--bounce-duration
--bounce-scale
--popover-bg
--shadow
--transition
```
