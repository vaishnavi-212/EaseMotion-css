# CSS Horizontal Roll Popover for Glassmorphism UI Layouts

A **pure CSS Horizontal Roll Popover** featuring a smooth rolling entrance animation with a modern **Glassmorphism UI** design. Built using only HTML and CSS, with responsive behavior and keyboard accessibility.

## Features

- Pure HTML & CSS
- Horizontal Roll animation
- Glassmorphism design
- Fully responsive
- Keyboard accessible (`:focus-within`)
- CSS Custom Properties
- No JavaScript
- Easy to customize
