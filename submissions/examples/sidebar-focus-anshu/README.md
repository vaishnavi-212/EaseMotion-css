# Sidebar Focus State Example

An interactive sidebar demonstration showcasing custom, highly visible `:focus-visible` animations designed for keyboard navigation.

## Key Features
- **High-Contrast Focus Indicators:** Focus rings utilizing custom shadow offsets that meet accessibility guidelines.
- **Micro-interactions:** Subtle element shifts (`transform`) optimized for both vertical (desktop) and horizontal (mobile) layouts.
- **Fluid Responsiveness:** Seamless transition from vertical sidebar to a clean mobile navigation bar.
- **Minimalist Code:** Pure HTML and CSS with zero external dependencies.

## Customization
You can adjust the color scheme or animation speeds inside `style.css` by modifying the values under the `:focus-visible` pseudo-class block.
