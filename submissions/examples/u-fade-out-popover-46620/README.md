# CSS Fade-Out Popover for Creative Portfolio Layouts

A **pure CSS Fade-Out Popover** designed for **Creative Portfolio Layouts**. The component provides a smooth fade-out transition, responsive layout, keyboard accessibility, and customizable CSS variables without requiring JavaScript.

## Features

- Pure HTML & CSS
- Smooth Fade-Out animation
- Creative Portfolio inspired design
- Fully responsive
- Keyboard accessible (`:focus-within`)
- CSS Custom Properties
- Lightweight
- No JavaScript
