# Ease Webhook Event Monitor

## Overview
A CSS-only component for displaying webhook delivery events.

## Features
- Delivery status badges
- Responsive layout
- Hover effects
- CSS variables
- Pure HTML & CSS

## Usage
Open `demo.html` in any browser.

## Browser Support
Chrome, Firefox, Edge, Safari.

## Why EaseMotion CSS?
A reusable animation-first component for API dashboards, webhook monitoring, and developer platforms.
