# Inline Alert Pulse

A compact alert row with a pulsing severity marker and stable readable copy.

## Files

- `demo.html` - inline alert markup.
- `style.css` - alert layout, marker pulse, and reduced-motion fallback.

## Highlights

- Uses `role="alert"` for the warning message.
- Keeps the pulse decorative with a pseudo-element.
- Stops the marker animation for reduced-motion users.
