# CSS Zoom-In Carousel – Modern SaaS

A pure CSS animated carousel featuring a smooth zoom-in interaction designed for Modern SaaS interfaces.

## ✨ Features

- Pure CSS
- Smooth zoom-in animation
- Responsive layout
- Keyboard accessible
- CSS custom properties
- No JavaScript required

## Usage

```html
<div class="carousel">
  <div class="card active">Dashboard</div>
  <div class="card">Analytics</div>
  <div class="card">Team</div>
</div>
```

## Custom Properties

```css
:root{
  --zoom-scale:1.1;
  --duration:.35s;
  --easing:cubic-bezier(.22,1,.36,1);
}
```

## Technologies

- HTML5
- CSS3

## Why EaseMotion CSS?

This component delivers a lightweight and reusable carousel for Modern SaaS dashboards using only HTML and CSS. It is responsive, accessible, customizable with CSS variables, and requires no JavaScript.
