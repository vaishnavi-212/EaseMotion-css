# Print-Optimized Alert Component

Alert/notification components with print media query optimization. Alerts render with simplified border styling when printed.

## Files

- `demo.html` - Alert demos with print optimization
- `style.css` - Alert styling with @media print rules

## Usage

Open `demo.html` in a browser to see the styled alerts. Use Ctrl+P (Cmd+P on Mac) to see simplified print styling.

## What This Does

Adds print media query optimization to alert components, removing backgrounds and shadows while keeping the border styling for readability on paper.

## Why Useful

Alert backgrounds use colored fills that waste ink when printed. This optimization ensures alerts remain readable with border styling while saving ink.
