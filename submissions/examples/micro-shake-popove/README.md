# CSS Micro Shake Popover

A responsive **Micro Shake Popover** for **Minimalist Tech Layouts**, built using pure HTML and CSS.

## Features

- Pure HTML & CSS
- Micro Shake hover animation
- Minimalist Tech UI
- Responsive Design
- Keyboard Accessible
- CSS Custom Properties
- No JavaScript

## Technologies

- HTML5
- CSS3

## Folder Structure

```
micro-shake-popover/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern browser.
