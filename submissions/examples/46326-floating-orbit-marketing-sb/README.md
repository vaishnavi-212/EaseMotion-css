# Floating Orbit Tooltip - Marketing Landing Page Layouts

A CSS tooltip component with floating orbit animation effect designed for marketing landing pages.

## Files

- `demo.html` - Demo page showing marketing landing page with tooltips
- `style.css` - Tooltip styles and animations

## Usage

Open `demo.html` in a browser and hover over navigation links, buttons, and stat labels to see the floating orbit tooltip animation.

## Features

### Floating Orbit Animation
- Tooltip gently floats with subtle rotation
- Blue gradient with shadow
- Smooth continuous animation loop

### Marketing Components
- Navigation bar with CTA
- Hero section with stats
- Feature cards with icons
- Landing page layout

### Design System
- Modern marketing aesthetic
- Dark hero section
- Blue accent color

## Classes

| Class | Description |
|-------|-------------|
| `ease-floating-orbit-mkt-sb` | Floating orbit tooltip for marketing |
| `ease-nav-link-sb` | Navigation link |
| `ease-nav-cta-sb` | Navigation CTA button |
| `ease-hero-btn-sb` | Hero primary button |
| `ease-hero-btn-outline-sb` | Hero outline button |
| `ease-feature-btn-sb` | Feature learn more button |

## Usage

Add `data-tooltip="Your text"` attribute to any element with `ease-floating-orbit-mkt-sb` class:

```html
<button class="ease-floating-orbit-mkt-sb" data-tooltip="Start your free trial">
  Get Started
</button>
```
