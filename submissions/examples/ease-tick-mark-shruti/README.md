# ease-tick-mark-animation

A reusable animated success tick component built using pure HTML, SVG, and CSS.

## Features

- Pure HTML, SVG & CSS
- Animated circular stroke
- Animated check mark
- Responsive design
- Lightweight and reusable
- Customizable through CSS variables
- No JavaScript required

## Files

```
demo.html

README.md
```

## CSS Custom Properties

```css
--ease-size
--ease-success
--ease-bg
--ease-page
--ease-shadow
--ease-duration
--ease-easing
```

## Accessibility

- Semantic SVG
- `role="img"` with `aria-label`
- Responsive layout
- Keyboard-friendly markup
