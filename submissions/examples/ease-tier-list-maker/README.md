# Ease Tier List Maker

A responsive Tier List Maker component built using pure HTML and CSS.

## Features

- Responsive layout
- Five ranking tiers (S, A, B, C, D)
- Hover animation
- Modern design
- Pure HTML & CSS
- Easy to customize

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in your browser to view the component.
