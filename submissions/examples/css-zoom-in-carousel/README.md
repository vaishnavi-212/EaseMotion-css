# CSS Zoom-In Carousel

A pure CSS animated carousel featuring a smooth zoom-in interaction with a Cyberpunk Neon design.

## ✨ Features

- Pure CSS
- Zoom-in hover animation
- Cyberpunk neon styling
- Keyboard accessible
- Responsive layout
- CSS custom properties
- No JavaScript required

## Usage

```html
<div class="carousel">
  <div class="card">Item</div>
</div>
```

## Custom Properties

```css
:root{
  --zoom-scale:1.12;
  --duration:.4s;
  --easing:cubic-bezier(.22,1,.36,1);
}
```

## Technologies

- HTML5
- CSS3

## Why EaseMotion CSS?

This component provides a reusable animation-first carousel with a neon-inspired interface using only CSS. It is lightweight, customizable, responsive, and accessible.
