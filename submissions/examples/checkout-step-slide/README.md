# Checkout Step Slide

This example adds a checkout progress card where each step slides into place with a stagger.

## Files

- `demo.html` contains the checkout progress markup.
- `style.css` defines the step states, staggered slide animation, responsive layout, and reduced-motion fallback.

## Highlights

- Uses an ordered list for step semantics.
- Shows done, active, and pending step states.
- Keeps each step at a stable height while animating.
- Switches to static states for reduced-motion users.
