# Range Slider Glide

This example adds a range slider whose thumb lifts gently on hover and keyboard focus.

## Files

- `demo.html` contains the labeled range input.
- `style.css` defines the slider track, thumb glide, focus state, and reduced-motion fallback.

## Highlights

- Uses a native range input with a visible label.
- Keeps the slider dimensions stable while the thumb animates.
- Adds hover and keyboard focus treatments.
- Removes thumb movement when reduced motion is requested.
