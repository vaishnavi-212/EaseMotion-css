# Roulette Wheel Spinner Component

A visually stunning CSS-only roulette wheel spinner component for promotional giveaways, contests, and interactive games.

## Files

- `demo.html` - Complete wheel markup with segments and controls
- `style.css` - Full styling with gradients, animations, and responsive design

## Usage

Open `demo.html` in any modern browser to interact with the wheel.

### HTML Classes

| Class | Description |
|-------|-------------|
| `.wheel-wrapper` | Container for pointer and wheel |
| `.wheel` | Outer wheel styling and shadow |
| `.wheel-inner` | Rotating segment container |
| `.segment` | Individual prize segments |
| `.wheel-center` | Center hub with icon |
| `.spin-btn` | Call-to-action button |
| `.prize-history` | Recent winners list |

## Features

- **Pure CSS implementation** - No JavaScript required for styling
- **Colorful segments** - 8 unique prize segments with distinct colors
- **Animated pointer** - Bouncing indicator animation
- **Pulsing center** - Glowing effect on the center hub
- **Responsive design** - Works on mobile and desktop
- **Winner history** - Shows recent prize winners

## Design Philosophy

This component captures the excitement of casino-style roulette wheels while maintaining accessibility and performance. The vibrant color scheme and smooth animations create an engaging experience perfect for promotional campaigns and interactive features.
