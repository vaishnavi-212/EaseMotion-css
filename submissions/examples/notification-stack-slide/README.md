# Notification Stack Slide

This example adds a compact notification stack where each item slides in with a short stagger.

## Files

- `demo.html` provides the notification list markup.
- `style.css` handles the staggered slide animation and reduced-motion fallback.

## Highlights

- Staggered motion across three notices.
- Stable card sizing to avoid layout jumps.
- Uses `aria-live="polite"` for notification semantics.
- Disables animation when users prefer reduced motion.
