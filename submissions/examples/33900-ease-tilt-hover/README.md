# Ease Tilt Hover

A smooth CSS-only 3D tilt hover animation for interactive UI elements.

## Description

Ease Tilt Hover adds a subtle three-dimensional tilt and lift effect when a user hovers over an element.

The animation smoothly rotates the element around the X and Y axes while slightly moving it upward. When the pointer leaves, the element smoothly returns to its original position.

## Features

- CSS-only implementation
- Smooth 3D perspective effect
- No JavaScript required
- Reusable for cards, buttons, and interactive UI elements
- Includes reduced-motion accessibility support
- Works by opening `demo.html` directly in a browser

## Usage

Add the animation class to an element:

```html
<div class="ease-tilt-hover">
  Hover over me
</div>
