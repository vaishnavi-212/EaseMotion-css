# Liquid Glass Floating Dock

A modern **CSS-only floating navigation dock** inspired by glassmorphism and desktop/mobile dock interfaces.

## Features

- ✨ Pure HTML & CSS
- 🪟 Frosted glass (Glassmorphism)
- 🚀 Smooth floating animation
- 🔥 Hover lift and magnification effect
- 📱 Responsive layout
- 🎯 No JavaScript required

## Folder Structure

```
liquid-glass-floating-dock/
├── demo.html
├── style.css
└── README.md
```

## Preview

The demo displays a floating glass navigation dock with animated icons that rise and slightly enlarge on hover.

## Technologies

- HTML5
- CSS3
- Flexbox
- CSS Animations
- Glassmorphism

## Usage

Open `demo.html` in any modern browser.

No installation or dependencies are required.
