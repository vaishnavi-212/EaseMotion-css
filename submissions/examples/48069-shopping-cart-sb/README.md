# Shopping Cart with Badge Component

A clean shopping cart interface with floating cart button and badge counter for e-commerce applications.

## Files

- `demo.html` - Shopping cart UI
- `style.css` - Full styling with animations

## Features

- Cart item list with quantity controls
- Floating cart button with badge
- Animated badge bounce effect
- Checkout summary
- Responsive layout

## Design Philosophy

Clean e-commerce cart interface with intuitive controls and prominent cart access.
