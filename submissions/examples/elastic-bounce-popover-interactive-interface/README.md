# CSS Elastic Bounce Popover – Interactive Interface

## Overview

A pure CSS Elastic Bounce Popover designed for modern Interactive Interface layouts. The popover smoothly expands with an elastic bounce animation while remaining responsive and keyboard accessible.

## Features

- Pure HTML & CSS
- Elastic Bounce animation
- Responsive layout
- Keyboard accessible
- CSS Custom Properties
- No JavaScript required

## Files

- demo.html
- style.css
- README.md

## Usage

```html
<div class="container">

    <button class="trigger-btn">
        Hover Me
    </button>

    <div class="popover">
        Interactive Content
    </div>

</div>
```

## Customization

```css
:root{
--duration:.55s;
--primary:#2563eb;
--popover:#1f2937;
--radius:14px;
}
```
