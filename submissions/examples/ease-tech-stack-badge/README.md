# Ease Tech Stack Badge

A lightweight CSS-only animated badge component for displaying technologies, frameworks, and tools.

## ✨ Features

- Pure HTML and CSS
- Smooth hover animation
- Responsive layout
- Keyboard focus support
- Customizable using CSS variables
- No JavaScript required

## 📌 Usage

Add the badge class:

```html
<span class="tech-badge">
  React
</span>
