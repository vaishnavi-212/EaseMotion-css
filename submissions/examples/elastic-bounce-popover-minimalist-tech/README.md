# CSS Elastic Bounce Popover – Minimalist Tech Layout

## Overview

A pure CSS Elastic Bounce Popover designed for Minimalist Tech interfaces. It delivers a clean bounce animation while remaining lightweight, responsive, and keyboard accessible.

## Features

- Pure HTML & CSS
- Elastic Bounce animation
- Responsive design
- Keyboard accessible
- CSS Custom Properties
- Zero JavaScript

## Files

- demo.html
- style.css
- README.md

## Usage

```html
<div class="wrapper">
    <button class="trigger-btn">Open Popover</button>

    <div class="popover">
        Minimalist Tech Popover
    </div>
</div>
```

## Customization

```css
:root{
    --primary:#111827;
    --duration:.55s;
    --radius:14px;
}
```
