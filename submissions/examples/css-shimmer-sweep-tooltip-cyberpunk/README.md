# CSS Shimmer Sweep Tooltip — Cyberpunk Neon Layouts

A bold, dark-themed pure CSS responsive tooltip module utilizing an animated linear-gradient reflection layer and rich neon parameter sets.

## Specifications
- **Mechanical Physics**: Runs a horizontal flashing sweep animation (`@keyframes sweep-flash`) that skews and passes a bright reflection highlight continuously over the card face on hover.
- **Declarative Operations**: Exposes configuration bounds (transition ease parameters, vertical offset intervals, reflection cycle times) cleanly inside central variable states.
- **Hardware Layering**: Couples translation pathways onto GPU compositor layers (`transform: translateY()`) to protect device drawing lanes from performance overhead.
