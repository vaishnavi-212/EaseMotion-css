# Particle Tab

## Overview

The **Particle Tab** is a lightweight, responsive tab component inspired by minimalist design principles. It provides a clean tabbed interface with smooth CSS animations and subtle particle-inspired visual effects while remaining completely functional without JavaScript.

The component is built using HTML and CSS, leveraging EaseMotion CSS variables, keyframes, and transitions to create an elegant and reusable UI element.

---

## Features

- ✨ Pure CSS implementation (No JavaScript required)
- 📑 Responsive tab navigation
- 🎨 Minimalist particle-inspired active indicator
- 🌈 Smooth transitions and EaseMotion animations
- 🎯 Keyboard accessible
- ⚙️ Customizable using CSS custom properties

---

## CSS Custom Properties

Customize the appearance of the component using the following variables:

| Variable | Description |
|----------|-------------|
| `--primary` | Active tab color |
| `--bg` | Background color |
| `--text` | Text color |
| `--duration` | Transition duration |

Example:

```css
:root {
    --primary: #6366f1;
    --bg: #ffffff;
    --text: #1f2937;
    --duration: 300ms;
}
```
