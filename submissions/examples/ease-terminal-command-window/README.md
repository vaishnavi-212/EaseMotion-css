# Ease Terminal Command Window

A reusable CSS-only terminal command window component for displaying code commands, CLI outputs, and developer-focused interfaces.

## Features

- Pure CSS implementation
- Responsive layout
- Terminal-style UI
- Smooth entrance animation
- Hover elevation effect
- Customizable using CSS variables

## Usage

Include the component:

```html
<div class="terminal-window">

    <div class="terminal-header">
        Terminal
    </div>

    <div class="terminal-body">
        <p>$ npm install package</p>
    </div>

</div>
