# Floating Orbit Tooltip – Interactive Interface

A pure CSS animated tooltip featuring a smooth Floating Orbit interaction designed for modern interactive interface layouts.

## Features

- Pure HTML and CSS
- Smooth tooltip entrance animation
- Floating orbit animation
- Multiple orbiting elements
- Responsive layout
- Keyboard accessible
- Customizable CSS custom properties
- Zero JavaScript
- Reduced-motion support
