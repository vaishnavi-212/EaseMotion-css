# CSS Zoom-Out Popover for Modern SaaS Layouts

A **pure CSS Zoom-Out Popover** created for **Modern SaaS Layouts**. The component features a smooth zoom-out animation, modern SaaS styling, responsive behavior, keyboard accessibility, and customizable CSS variables without JavaScript.

## Features

- Pure HTML & CSS
- Smooth Zoom-Out animation
- Modern SaaS design
- Fully responsive
- Keyboard accessible (`:focus-within`)
- CSS Custom Properties
- Lightweight
- No JavaScript required
