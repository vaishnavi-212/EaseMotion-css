# Ripple Color Picker

A ripple-style color picker inspired by neon design patterns for EaseMotion CSS.

## Preview

Open `demo.html` in your browser to view the component.

## Features

- Pure HTML & CSS
- No JavaScript
- Neon-inspired design
- Ripple selection animation
- Responsive layout
- Keyboard accessible
- Uses EaseMotion animation classes

## Folder Structure

```
ripple-color-picker-sr/
├── demo.html
├── style.css
└── README.md
```

## Usage

Include the component stylesheet.

```html
<link rel="stylesheet" href="style.css">
```

Open `demo.html` in any modern browser.

## Accessibility

- Uses semantic radio inputs
- Keyboard navigation supported
- Focus-visible styles included

## Browser Support

- Chrome
- Edge
- Firefox
- Safari
