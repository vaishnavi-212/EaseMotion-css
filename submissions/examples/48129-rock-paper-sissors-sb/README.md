# Rock Paper Scissors Game Component

An interactive CSS-only Rock Paper Scissors game with animated hand choices, score tracking, and match history. Perfect for gaming sections and interactive entertainment features.

## Files

- `demo.html` - Game interface with choices and scoreboard
- `style.css` - Full styling with animations and responsive design

## Usage

Open `demo.html` in any modern browser to play the game.

### HTML Classes

| Class | Description |
|-------|-------------|
| `.scoreboard` | Score display for player and computer |
| `.battle-arena` | Area showing player and computer choices |
| `.choice-hand` | Hand emoji display |
| `.game-btn` | Rock, Paper, Scissors choice buttons |
| `.match-history` | List of recent match results |

## Features

- **Pure CSS styling** - No JavaScript required for visuals
- **Animated hands** - Visual feedback on choices
- **Score tracking** - Real-time score display
- **Match history** - Shows recent game results
- **Responsive design** - Works on mobile and desktop
- **Glowing hover effects** - Color-coded button highlights

## Design Philosophy

This component brings the classic Rock Paper Scissors game to life with a modern gaming aesthetic. The dark theme creates an arcade-like atmosphere, while the glowing hover effects and smooth animations make the experience engaging and fun.
