# CSS-Only Tabs Component

A fully functional, animated tabbed interface that relies entirely on CSS, using the native HTML `<input type="radio">` and `<label>` pattern. It features smooth fade animations when switching panels and utilizes EaseMotion tokens for styling.

## Files
- `demo.html` - Interactive demonstration of the tabs component.
- `style.css` - The CSS implementation utilizing EaseMotion tokens.

## Details
This addresses the "CSS-only tabs" requirement currently listed as "Planned" in the project Roadmap. It requires zero JavaScript to operate and is completely keyboard accessible.
