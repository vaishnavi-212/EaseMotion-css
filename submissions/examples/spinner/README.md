# ease-spinner

## What does this do?

`ease-spinner` provides a collection of pure CSS loading spinners and loading indicators for applications, forms, dashboards, and asynchronous actions.

The component includes multiple spinner styles such as:

- Classic rotating ring
- Bouncing dots
- Equalizer bars
- Dual rotating rings
- Size variants
- Color variants

No JavaScript is required.
