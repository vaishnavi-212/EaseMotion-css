# Color Shift Pricing Card

A modern pricing card with a smooth color-shifting background animation. It is designed for creative agencies, portfolios, SaaS products, and landing pages while keeping the implementation simple with pure HTML and CSS.

---

## Features

- Pure HTML and CSS
- No JavaScript required
- Animated color-shift gradient
- Responsive design
- Modern creative agency style
- Easy to customize
- Compatible with EaseMotion CSS utility classes

## How to Use

Include the following markup in your HTML:

```html
<section class="pricing-section ease-fade-in">

    <div class="pricing-card ease-hover-lift">

        <div class="card-header">
            <span class="plan-tag">Creative Plan</span>

            <h1>$29 <small>/month</small></h1>
        </div>

        <ul class="feature-list">
            <li>Unlimited Projects</li>
            <li>Team Collaboration</li>
            <li>Premium Templates</li>
        </ul>

        <a href="#" class="pricing-btn">
            Get Started
        </a>

    </div>

</section>
```

Open **demo.html** in any modern browser to view the component.

---

## Why This Component?

Pricing cards are one of the most common UI components on modern websites. This version uses a smooth animated color-shift effect to highlight the pricing plan without being distracting. It is suitable for creative agencies, SaaS products, startup landing pages, portfolios, and business websites.
