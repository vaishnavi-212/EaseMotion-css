# Magnetic Pull Tooltip – Cyberpunk Neon

A pure CSS Magnetic Pull Tooltip inspired by Cyberpunk Neon interfaces.

## Features

- Pure CSS
- No JavaScript
- Smooth magnetic pull animation
- Neon glow styling
- Responsive
- Keyboard accessible
- Uses CSS Custom Properties

## Accessibility

- Keyboard focus support
- Visible focus outline
- Semantic tooltip (`role="tooltip"`)
- Responsive layout
