
# Ease Rate Check 150

A responsive rating selector with animated check confirmation states.



## How to use

Open `demo.html` directly in a browser. Select any rating card to see the checked state and animated confirmation feedback.



```html

<label class="rate-option">

  <input type="radio" name="rating" value="5" />

  <span class="rate-card">

    <span class="rate-number">5</span>

    <span class="rate-label">Excellent</span>

  </span>

</label>
```

## Why it is useful

This component gives feedback forms, product reviews, surveys, and onboarding flows a polished interaction while staying lightweight, responsive, keyboard-friendly, and dependency-free.
