# Print-Optimized Accordion Component

Accordion component with print media query optimization. All accordion sections expand when printed for better readability.

## Files

- `demo.html` - Accordion demo with print optimization
- `style.css` - Accordion styling with @media print rules

## Usage

Open `demo.html` in a browser to see the accordion. Use Ctrl+P (Cmd+P on Mac) to see all content expanded in print mode.

## What This Does

Adds print media query optimization to accordion components:
- Expands all accordion sections when printing
- Removes decorative icons
- Ensures all content is visible on paper

## Why Useful

Accordions hide content by default, which is not ideal for printing. This optimization ensures all accordion content is visible on printed pages for complete documentation.
