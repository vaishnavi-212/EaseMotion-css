# Magnetic Pull Tooltip – Modern SaaS Layout

**Issue:** #46296

## Description

A pure CSS **Magnetic Pull Tooltip** designed for Modern SaaS interfaces. The tooltip smoothly animates toward the trigger element, creating a subtle magnetic pull effect without JavaScript.

## Features

- Pure HTML & CSS
- Magnetic Pull animation
- Modern SaaS design
- Responsive layout
- Keyboard accessible
- CSS Custom Properties
- No JavaScript required

## Usage

```html
<div class="tooltip-wrapper">
    <button class="cta-btn">Start Free Trial</button>
    <span class="tooltip">No credit card required 🚀</span>
</div>
```

Include:

```html
<link rel="stylesheet" href="style.css">
```

## Customization

```css
:root{
    --primary:#4f46e5;
    --secondary:#7c3aed;
    --duration:.35s;
    --pull-distance:18px;
}
```
