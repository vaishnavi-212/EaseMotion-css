# Shimmer Sweep Tooltip - Product Catalog Layouts

## Overview

A pure CSS animated tooltip featuring a smooth shimmer sweep interaction designed for Product Catalog interfaces.

The component enhances product cards with a modern animated information preview without JavaScript.

## Features

- Pure CSS animation
- Smooth shimmer sweep effect
- Product card friendly design
- Responsive layout
- Keyboard accessible
- CSS variable customization
- No JavaScript required

## Usage

```html
<div class="tooltip-wrapper">

<button class="product-btn">
 View Details
</button>

<div class="tooltip">
 Product information
</div>

</div>
