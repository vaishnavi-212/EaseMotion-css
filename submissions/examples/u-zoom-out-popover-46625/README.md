# CSS Zoom-Out Popover for Product Catalog Layouts

A **pure CSS Zoom-Out Popover** designed for **Product Catalog Layouts**. It provides a smooth zoom-out entrance animation with a clean product-focused interface, built entirely with HTML and CSS.

## Features

- Pure HTML & CSS
- Smooth Zoom-Out animation
- Product Catalog UI
- Fully responsive
- Keyboard accessible (`:focus-within`)
- CSS Custom Properties
- Lightweight
- No JavaScript required
