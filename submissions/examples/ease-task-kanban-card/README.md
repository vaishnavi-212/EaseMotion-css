# ease-task-kanban-card

Issue: #49948

## Overview

A clean Kanban-style task card component for project management dashboards.

It visually represents tasks in:

- To Do
- In Progress
- Done

The component is visual-only and follows EaseMotion CSS styling principles.

---

## Features

- Responsive Kanban layout
- Smooth hover lift animation
- Priority badges
- Status colors
- Task title & description
- Assignee information
- Due date section
- Pure HTML & CSS
- No JavaScript
