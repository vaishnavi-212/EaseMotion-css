# Rolling Dice Animation Component

A beautifully designed CSS-only dice component with realistic 3D styling and smooth animations. Perfect for board game interfaces, gambling apps, or random selection features.

## Files

- `demo.html` - Complete dice board with all 6 faces
- `style.css` - Full styling with 3D effects and animations

## Usage

Open `demo.html` in any modern browser to view the animated dice.

### HTML Classes

| Class | Description |
|-------|-------------|
| `.dice` | Base dice container |
| `.dice-X` | Specific face number (1-6) |
| `.dot` | Individual dot on dice face |
| `.center` | Center dot position |
| `.top-left`, `.top-right` | Top corner dots |
| `.bottom-left`, `.bottom-right` | Bottom corner dots |
| `.middle-left`, `.middle-right` | Middle side dots |

## Features

- **Pure CSS implementation** - No JavaScript for visuals
- **Realistic 3D styling** - Gradient shadows and depth effects
- **Animated entrance** - Dice bounce in with rotation
- **Interactive hover** - Lift effect on hover
- **All 6 faces** - Complete dice face library
- **Statistics display** - Roll count and average tracking

## Design Philosophy

This component brings the tactile satisfaction of rolling real dice to the web. The 3D styling with gradients and shadows creates depth, while the bouncy animations add playfulness appropriate for game-related interfaces.
