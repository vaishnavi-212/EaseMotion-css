# CSS Neon Glow Popover

A pure CSS Neon Glow Popover designed for Interactive Interface layouts.

## Features

- Pure HTML & CSS
- Neon Glow Animation
- Interactive Interface Design
- Responsive Layout
- Keyboard Accessible
- CSS Custom Properties
- No JavaScript

## Technologies

- HTML5
- CSS3

## Folder Structure

```
neon-glow-popover/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern browser.
