# ease-split-button-menu

A reusable Split Button Menu built with pure HTML and CSS.

## Features

- Pure HTML + CSS
- No JavaScript
- Uses `<details>` and `<summary>`
- Responsive
- Keyboard accessible
- Dropdown menu animation
- CSS custom properties

## Files

```
demo.html
style.css
README.md
```
