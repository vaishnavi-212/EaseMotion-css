# Custom Scrollbar Component

Stylized scrollbar components with smooth animations for enhanced user experience.

## Files

- `demo.html` - Demo page showing various scrollbar styles
- `style.css` - Scrollbar styles and animations

## Usage

Open `demo.html` in a browser and scroll within each container to see the custom scrollbar styles.

## Features

### Scrollbar Variants
- Default scrollbar
- Thin scrollbar
- Round scrollbar with gradient
- Primary color scrollbar

### Styling Options
- Custom track and thumb colors
- Rounded corners
- Hover effects
- Smooth animations
- Firefox support

## Classes

| Class | Description |
|-------|-------------|
| `ease-scroll-container-sb` | Default styled scrollbar |
| `ease-scroll-thin-sb` | Thin minimal scrollbar |
| `ease-scroll-round-sb` | Rounded gradient scrollbar |
| `ease-scroll-primary-sb` | Primary color scrollbar |

## Browser Support

- Webkit browsers (Chrome, Safari, Edge)
- Firefox (via scrollbar-width and scrollbar-color)

## Usage

Apply a scrollbar class to any scrollable container:

```html
<div class="ease-scroll-container-sb">
  Scrollable content here
</div>
```
