# Print-Optimized Modal Component

A modal/dialog component with print media query optimization for clean printing without backgrounds, shadows, or unnecessary elements.

## Files

- `demo.html` - Modal markup with print optimization
- `style.css` - Modal styling with @media print rules

## Usage

Open `demo.html` in a browser and click "Open Modal" to see the modal. Use Ctrl+P (Cmd+P on Mac) to see the print-optimized version.

## What This Does

Adds print media query optimization to modal components, removing overlay backgrounds, shadows, and the close button for clean printed output.

## Why Useful

Modal dialogs often have dark overlay backgrounds and decorative shadows that waste ink when printed. This optimization ensures modals render cleanly on paper.
