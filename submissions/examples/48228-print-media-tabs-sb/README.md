# Print-Optimized Tabs Component

Tabs component with print media query optimization. All tab panels are visible when printed.

## Files

- `demo.html` - Tabs demo with print optimization
- `style.css` - Tabs styling with @media print rules

## Usage

Open `demo.html` in a browser to see the tabs. Use Ctrl+P (Cmd+P on Mac) to see all content expanded in print mode.

## What This Does

Adds print media query optimization to tabs components:
- Shows all tab panels when printing
- Removes active indicator styling
- Ensures all content is visible on paper

## Why Useful

Tabs hide content by default, which is not ideal for printing. This optimization ensures all tab content is visible on printed pages.
