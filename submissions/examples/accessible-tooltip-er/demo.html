Accessible Web Tooltip

CSS Zoom-Out Tooltip for Accessible Interfaces

Why Accessibility Matters in Tooltips

This tooltip follows WCAG 2.1 guidelines with keyboard support, proper ARIA attributes, and high contrast. Navigate using Tab to reveal tooltips — no hover required.

Interactive Demo

Contrast WCAG AAA 7:1 contrast ratio for text
Focus Order Logical tab order follows visual layout
Motion Respects prefers-reduced-motion setting
Keyboard Fully operable without a mouse
Scalable Text scales up to 200% without layout break
Color Never relies on color alone to convey info

High Contrast Theme

Screen Reader Semantic HTML ensures screen reader compatibility
Target Size Minimum 44×44px touch targets per WCAG 2.5.5
Reflow Content reflows at 320px without horizontal scroll

Usage

<span tabindex="0" role="button" aria-describedby="tip-id">
    Hover or Focus Me
</span>
<span class="tooltip tooltip--accessible" role="tooltip" id="tip-id">
    <span class="tooltip__content">Your tooltip text</span>
</span>

Custom Properties

Property Default Description
--ez-tooltip-scale 0.3 Initial scale for zoom-out
--ez-tooltip-duration 0.3s Animation duration
--ez-tooltip-offset 12px Gap from trigger
--ez-tooltip-bg #1a1a2e Background color
--ez-tooltip-color #ffffff Text color
