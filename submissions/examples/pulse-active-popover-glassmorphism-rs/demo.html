Glassmorphism Popover

Pure CSS pulse-active popover with glassmorphism aesthetic — click any card to open

✨

Feature Info

Glassmorphism popover with backdrop blur and gradient accents

✨ Feature Details

This glassmorphism popover uses backdrop-filter: blur() for the frosted glass effect with a semi-transparent background and subtle border.

The pulse animation is fully customizable via CSS custom properties.

🎨

Color Scheme

Gradient accents and glow effects complement the glass surface

🎨 Color Palette

Primary: #6c63ff with glowing shadow

Surface: rgba(255,255,255,0.08) with blur

Border: rgba(255,255,255,0.15)

⚡

Pulse Animation

Smooth pulse-active transition with customizable timing and scale

⚡ Pulse Settings

Duration: 2s cycle

Scale: 1.04x at peak

Easing: ease-in-out

Override with --popover-pulse-duration and --popover-pulse-scale.

🛡️

Accessibility

Keyboard navigable with reduced-motion and forced-colors support

🛡️ Accessibility

✓ Keyboard accessible via Tab and Enter/Space

✓ prefers-reduced-motion disables animations

✓ forced-colors adapts to high-contrast mode

✓ prefers-color-scheme switches light/dark
