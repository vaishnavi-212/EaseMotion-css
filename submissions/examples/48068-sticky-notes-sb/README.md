# Sticky Notes Grid Component

A playful sticky notes grid with colorful notes, pin decorations, and hover animations. Perfect for notes apps, reminders, or bulletin boards.

## Files

- `demo.html` - Sticky notes display
- `style.css` - Full styling with animations

## Features

- Multiple color variants (yellow, pink, blue, green)
- Pin decorations
- Hover lift effects
- Drop animation on load
- Handwritten-style font

## Design Philosophy

Fun, playful sticky notes that feel tactile and engaging with realistic shadows and pins.
