# CSS 3D Perspective Tilt Popover for Dashboard Analytics Layouts

A pure CSS animated **3D Perspective Tilt Popover** designed for modern Dashboard Analytics interfaces.

## Features

- Pure HTML & CSS
- Smooth 3D Perspective Tilt animation
- Responsive layout
- Dashboard Analytics inspired design
- Keyboard accessible
- CSS Custom Properties
- No JavaScript
- Reusable component
