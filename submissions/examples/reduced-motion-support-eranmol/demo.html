prefers-reduced-motion Support

Issue #47376 — Accessibility enhancement for motion-sensitive users

Enable "Reduce Motion" in your OS settings to see the effect:
Windows: Settings → Accessibility → Visual effects → Animation effects
macOS: System Preferences → Accessibility → Display → Reduce motion

Animated Elements (with motion)

Pulse
Bounce
Spin
Fade

Hover Interactions

Scroll-triggered Animation

This element animates when it enters the viewport.

Implementation

/* Add to your CSS file */
@media (prefers-reduced-motion: reduce) {
  *,
  *::before,
  *::after {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
    scroll-behavior: auto !important;
  }
}

Using 0.01ms instead of none preserves the final state of animations while effectively disabling motion.
