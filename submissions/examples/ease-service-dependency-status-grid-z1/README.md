# Ease Service Dependency Status Grid

## Overview
A CSS-only grid component for visualizing microservice dependency health.

## Features
- Responsive service grid
- Animated health indicators
- Hover interactions
- CSS variables
- Pure HTML & CSS

## Folder Structure
demo.html
style.css
README.md

## Usage
Open `demo.html` directly in any modern browser.

## Browser Support
Chrome, Firefox, Edge, Safari

## Responsive
Optimized for desktop, tablet, and mobile devices.

## Why EaseMotion CSS?
Provides a reusable animation-first component for infrastructure dashboards, cloud monitoring platforms, and microservice architecture visualization.
