# Ease Teacher Grade Card

A reusable CSS-only teacher grade card component for displaying
student performance, subject details, and academic statistics.

## Features

- Pure HTML and CSS
- Smooth hover animation
- Modern card UI
- Responsive layout
- Easy customization
- No JavaScript required

## Usage

```html
<div class="grade-card">
  <h2>Mathematics</h2>
  <strong>A+</strong>
</div>
