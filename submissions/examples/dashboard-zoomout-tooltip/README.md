# CSS Zoom-Out Tooltip for Dashboard Analytics Layouts

A responsive **pure CSS Zoom-Out Tooltip** designed for dashboard analytics interfaces, statistics cards, and data visualization layouts.

---

## Features

- Pure HTML & CSS
- Smooth Zoom-Out animation
- Dashboard analytics inspired UI
- CSS Custom Properties for easy customization
- Responsive layout
- Keyboard accessible (`:focus-within`)
- No JavaScript required

## Customization

The following CSS variables can be modified inside `:root`:

```css
--tooltip-bg
--tooltip-text
--tooltip-width
--tooltip-radius
--tooltip-duration
--tooltip-easing
--tooltip-scale
--primary
--success
```

---

## Accessibility

- Keyboard accessible using `:focus-within`
- Uses `aria-describedby`
- Uses `role="tooltip"`
- Semantic HTML structure

---

## Preview

The demo showcases a dashboard analytics card with a help button. Hovering or focusing on the button displays a smooth Zoom-Out tooltip containing analytics insights and update information.

Works in all modern browsers supporting:
