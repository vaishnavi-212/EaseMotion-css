# Button Component

A comprehensive button component with multiple variants, sizes, and states.

## Files

- `demo.html` - Button examples
- `style.css` - Full styling with animations

## Features

- Multiple variants (primary, secondary, outline, ghost)
- Multiple sizes (small, medium, large)
- Icon button support
- Disabled and loading states
- Hover animations

## Design Philosophy

Versatile, accessible buttons with satisfying hover effects and clear visual states.
