# Avatar Component

A clean avatar component with multiple sizes, image support, and avatar groups.

## Files

- `demo.html` - Avatar examples
- `style.css` - Full styling

## Features

- Multiple sizes (sm, md, lg, xl)
- Image avatars
- Initials avatars
- Avatar group with overlap

## Design Philosophy

Clean, circular avatars for user profiles and group displays.
