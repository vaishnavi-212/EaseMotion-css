# Right Arrow Icon Component

A versatile right arrow icon with multiple animation variants for navigation and UI components.

## Files

- `demo.html` - Demo page showing all arrow icon variants
- `style.css` - Arrow icon styles and animations

## Usage

Open `demo.html` in a browser to see the arrow icons in action.

## Features

### Animation Variants

1. **Default** - Simple hover effect with shadow
2. **Bounce** - Arrow bounces on hover
3. **Slide** - Arrow slides right on hover
4. **Grow** - Arrow scales up on hover

### Size Variants

- `ease-arrow-sm` - Small (40px)
- `ease-arrow-md` - Medium (48px, default)
- `ease-arrow-lg` - Large (64px)

### Navigation Link

Use `ease-nav-link-sb` with `ease-arrow-inline-sb` for inline arrow navigation buttons.

## Classes

| Class | Description |

| `ease-arrow-btn` | Button container for arrow icon |
| `ease-arrow-icon` | The arrow triangle shape |

| `ease-arrow-right-bounce-sb` | Bounce animation on hover |
| `ease-arrow-right-slide-sb` | Slide animation on hover |
| `ease-arrow-right-grow-sb` | Grow animation on hover |
| `ease-arrow-sm` | Small size |
