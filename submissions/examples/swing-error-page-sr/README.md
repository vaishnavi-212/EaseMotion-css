# Swing Error Page

A clean and responsive 404 error page with a smooth swing animation. It is designed for corporate websites, business portals, and SaaS applications using only HTML and CSS.

---

## Features

- Pure HTML and CSS
- No JavaScript required
- Smooth swing animation
- Corporate-inspired design
- Fully responsive layout
- Accessible structure
- Compatible with EaseMotion CSS utility classes

## How to Use

Add the following markup to your HTML:

```html
<section class="error-page ease-fade-in">

    <div class="error-card ease-hover-lift">

        <div class="error-icon">

            <div class="error-circle">
                <h1>404</h1>
            </div>

        </div>

        <div class="error-content">

            <h2>Page Not Found</h2>

            <p>
                Sorry, the page you are looking for could not be found.
            </p>

            <a href="#" class="primary-btn">
                Go to Homepage
            </a>

        </div>

    </div>

</section>
```

Open **demo.html** in any modern browser to view the component.

---

## Why This Component?

Error pages are an important part of user experience. This component adds a subtle swing animation to the error illustration, making the page feel more engaging while maintaining a clean and professional corporate appearance. Since it is built with pure HTML and CSS, it is lightweight, reusable, and easy to integrate.
