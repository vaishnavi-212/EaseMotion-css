# Search Filter Chip

This example adds a search field with filter chips and a subtle selected-chip pulse.

## Files

- `demo.html` contains the search input and filter chip controls.
- `style.css` defines chip hover, focus, selected pulse, and reduced-motion behavior.

## Highlights

- Uses real buttons for chip controls.
- Keeps chip height stable during hover and selected states.
- Includes visible keyboard focus styling.
- Stops animation for users who prefer reduced motion.
