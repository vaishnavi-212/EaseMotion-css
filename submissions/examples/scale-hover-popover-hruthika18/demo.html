Scale-Hover Popover

Pure CSS popover with a crisp scale-up entrance, styled for minimalist tech UIs. Hover or tab to any trigger below.

API_TIMEOUT ⓘ Max time in ms before a request is aborted. Default: 8000.
MAX_RETRIES ⓘ Number of retry attempts on transient network failures.
CACHE_TTL ⓘ Time-to-live for cached responses, in seconds. Set to 0 to disable caching.

Keyboard users: press Tab to focus a trigger and reveal its popover. Press Tab again to move on.
