# Fade-In Popover - Marketing Landing Page

## Overview

A pure CSS animated Fade-In Popover component designed for modern marketing landing page interfaces.

The component provides a smooth appearance animation without using JavaScript.

## Features

- Pure CSS animation
- Fade-in interaction effect
- Responsive design
- Keyboard accessible using focus-within
- Customizable CSS variables
- Lightweight and reusable

## Usage

```html
<div class="popover">

<button class="popover-btn">
  Product Features
</button>

<div class="popover-content">
  Your content here
</div>

</div>
