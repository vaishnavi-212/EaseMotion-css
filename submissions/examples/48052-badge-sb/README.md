# Badge Component

A comprehensive badge component with multiple styles, colors, and variants.

## Files

- `demo.html` - Badge examples
- `style.css` - Full styling

## Features

- Multiple colors (primary, success, warning, danger, info)
- Solid and outline styles
- Pill shape variant
- Count badges with numbers

## Design Philosophy

Clean, informative badges for labeling, categorizing, and showing counts.
