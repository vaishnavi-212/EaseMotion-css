# Neon Glow Popover – Minimalist Tech

Issue: #46486

## Overview

A pure CSS Neon Glow Popover designed for Minimalist Tech interfaces. It features a smooth glow transition without requiring JavaScript.

## Features

- Pure CSS
- Responsive
- Keyboard Accessible
- CSS Custom Properties
- Smooth Neon Glow Animation
- Lightweight

## CSS Variables

```css
:root{
  --glow-color:#00f5ff;
  --duration:.45s;
  --scale:.95;
}
```

## Usage

```html
<div class="popover">
    <button class="trigger">Hover Me</button>

    <div class="popover-content">
        Popover Content
    </div>
</div>
```
