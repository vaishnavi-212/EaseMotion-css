# CSS Fade-Out Popover for Product Catalog Layouts

A **pure CSS Fade-Out Popover** built for **Product Catalog Layouts**. This component features a smooth fade-out animation, responsive product card design, keyboard accessibility, and customizable CSS variables without using JavaScript.

## Features

- Pure HTML & CSS
- Smooth Fade-Out animation
- Product Catalog inspired design
- Fully responsive
- Keyboard accessible (`:focus-within`)
- CSS Custom Properties
- Lightweight
- No JavaScript required
