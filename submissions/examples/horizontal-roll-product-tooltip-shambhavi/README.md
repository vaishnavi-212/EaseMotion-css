# Horizontal Roll Tooltip – Product Catalog

A responsive Product Catalog tooltip featuring a smooth Horizontal Roll animation built entirely with HTML and CSS.

## Features

- Pure HTML & CSS
- Horizontal Roll animation
- Product Catalog inspired design
- Responsive layout
- Keyboard accessible
- CSS custom properties
- Zero JavaScript
