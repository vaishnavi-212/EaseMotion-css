# Pill Toggle Snap

This example adds a segmented pill toggle with a sliding selection indicator.

## Files

- `demo.html` contains the two-option toggle markup.
- `style.css` defines the pill indicator, hover and focus states, and reduced-motion fallback.

## Highlights

- Uses real buttons inside a labeled control group.
- Keeps the pill dimensions stable while the indicator moves.
- Supports hover and keyboard focus states.
- Removes indicator transition timing for reduced-motion users.
