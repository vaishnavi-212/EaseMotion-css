# Print-Optimized Skeleton Component

Skeleton/loading placeholder components with print media query optimization. Skeletons are hidden when printing.

## Files

- `demo.html` - Skeleton demo with print optimization
- `style.css` - Skeleton styling with @media print rules

## Usage

Open `demo.html` in a browser to see the animated skeleton screens. Use Ctrl+P (Cmd+P on Mac) to see skeletons hidden in print mode.

## What This Does

Adds print media query optimization to skeleton components, hiding them when printing. Skeletons are loading placeholders that have no meaning on printed pages.

## Why Useful

Skeleton screens are purely visual loading indicators that serve no purpose when printed. Hiding them produces cleaner printed output.
