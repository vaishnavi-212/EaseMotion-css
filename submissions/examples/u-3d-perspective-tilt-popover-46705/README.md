# CSS 3D Perspective Tilt Popover for Responsive Dashboard Layouts

A pure CSS animated **3D Perspective Tilt Popover** designed for responsive dashboard interfaces. The popover smoothly tilts into view using CSS transforms and perspective, creating a modern interactive experience without any JavaScript.

## Features

- 🎨 Pure HTML & CSS
- 🚀 Smooth 3D Perspective Tilt animation
- 📱 Fully responsive design
- ♿ Keyboard accessible using `:focus-within`
- 🎛 Easily customizable with CSS Custom Properties
- 💡 Modern dashboard-inspired UI
- ⚡ Lightweight and reusable
- ❌ No JavaScript required

## CSS Custom Properties

```css
:root{
    --primary:#4f7cff;
    --transition:.55s;
    --ease:cubic-bezier(.22,1,.36,1);
    --scale:1.03;
    --rotateX:12deg;
    --rotateY:-12deg;
}
```

These variables allow you to customize:

- Animation duration
- Animation easing
- Scale effect
- 3D rotation angles
- Primary color
