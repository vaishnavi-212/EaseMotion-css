# Ease Taxi Fare Meter

A responsive Taxi Fare Meter component built with pure HTML and CSS.

## Features

- Digital fare display
- Distance and time information
- Modern taxi meter UI
- Responsive design
- Hover button animations
- Pure HTML & CSS
- No JavaScript required

## Folder Structure

```
ease-taxi-fare-meter/
├── demo.html
├── style.css
└── README.md
```

## Usage

Include the stylesheet:

```html
<link rel="stylesheet" href="style.css">
```

Open `demo.html` directly in your browser.

## Customization

- Change fare color in `.display`
- Modify animation speed in `@keyframes blink`
- Update fare, distance, and time values in `demo.html`
- Customize button colors and fonts

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
