# Scatter Plot Chart Component

A pure CSS scatter plot chart component for visualizing correlations between two variables. Perfect for data dashboards and analytics platforms.

## Files

- `demo.html` - Scatter plot with data points, trend line, and statistics
- `style.css` - Full styling including animations and responsive design

## Usage

Open `demo.html` in any modern browser to view the interactive scatter plot chart.

### HTML Classes

| Class | Description |
|-------|-------------|
| `.chart-container` | Main container for the chart |
| `.chart-area` | SVG/div area containing data points |
| `.data-point` | Individual data point markers |
| `.trend-line` | Linear regression trend line |
| `.grid-lines` | Background grid for reference |
| `.chart-stats` | Statistical summary cards |
| `.chart-legend` | Legend explaining chart elements |

## Features

- **Pure CSS implementation** - No JavaScript required
- **Interactive tooltips** - Hover/focus on data points to see values
- **Animated entrance** - Data points pop in with staggered animation
- **Trend line** - Visual indicator of correlation direction
- **Statistics cards** - Correlation, R² score, and data point count
- **Responsive design** - Adapts to mobile screens

## Design Philosophy

This component embodies EaseMotion's principle of making data visualization both informative and visually engaging. The scatter plot uses subtle animations to draw attention while maintaining clarity and readability essential for analytical content.
