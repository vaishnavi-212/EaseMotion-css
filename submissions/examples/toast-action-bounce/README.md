# Toast Action Bounce

A toast message with an action button that bounces gently on hover or keyboard focus.

## Files

- `demo.html` - toast status and action button markup.
- `style.css` - button bounce, focus-visible state, and reduced-motion fallback.

## Highlights

- Uses `role="status"` for the toast content.
- Mirrors hover feedback on keyboard focus.
- Disables animation for reduced-motion users.
