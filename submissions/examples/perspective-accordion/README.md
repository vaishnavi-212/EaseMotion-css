# Perspective Accordion

A modern **Perspective Accordion** component built entirely with **HTML** and **CSS**, inspired by Cyberpunk design patterns. The component features a subtle 3D perspective effect, smooth panel transitions, responsive layouts, and accessibility support—all without requiring JavaScript.

---

## Features

- Pure HTML & CSS
- Zero JavaScript
- Cyberpunk-inspired design
- Smooth perspective opening animation
- Native HTML accordion using `<details>` and `<summary>`
- Responsive layout
- Keyboard accessible
- Supports `prefers-reduced-motion`
- Easily customizable using CSS

## Customization

You can easily customize the component by modifying the colors, border styles, spacing, and animation timing in `style.css`.

Common customizations include:

- Accent colors
- Background colors
- Border radius
- Perspective depth
- Animation duration
- Typography
- Shadows

---

## Usage

```html
<link rel="stylesheet" href="style.css">

<details open>

    <summary>
        Neural Interface
    </summary>

    <div class="content">
        Connect directly with next-generation neural hardware using secure encrypted channels.
    </div>

</details>
```

Open **demo.html** directly in any modern browser to preview the component.
