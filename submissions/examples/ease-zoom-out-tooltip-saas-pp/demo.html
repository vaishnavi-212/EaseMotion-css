CSS Zoom-Out Tooltip for Modern SaaS

A pure CSS animated tooltip that settles into place with a smooth zoom-out transition, styled for modern SaaS interfaces. Fully responsive, keyboard accessible, and tunable through standard CSS custom properties — zero JavaScript required for the animation.

Accessibility: The trigger wrapper uses tabindex="0" and :focus-within, so the tooltip is reachable by keyboard. role="tooltip" is set on each bubble, and prefers-reduced-motion disables the zoom for users who request reduced motion.

Parameter Controls

These sliders mutate the CSS custom properties that drive the zoom-out transition. Changes apply live to every tooltip on the page.
