# Ease Image Print Optimization

## Overview

This example demonstrates how the **ease-image** component can be optimized for printing.

The print stylesheet removes decorative backgrounds, gradients, shadows, and unnecessary visual effects while improving text readability and reducing ink usage.

---

## Features

- Print-friendly styling
- Removes gradients
- Removes box shadows
- Uses transparent backgrounds
- Converts text to black
- Improves readability on paper
- Saves printer ink

## Installation

Open `demo.html` in any modern browser.

Press **Ctrl + P** (Windows/Linux) or **Cmd + P** (macOS) to view the print preview.

---

## Print Optimizations

The following styles are applied inside:

```css
@media print
```

- Remove gradients
- Remove shadows
- Set background to transparent
- Change text color to black
- Add a simple border for structure
