\# ✍️ Typewriter Select Menu Component



A polished, accessible select menu component with a typewriter-inspired animation effect for the EaseMotion CSS library.



\## 📸 Demo



!\[Typewriter Select Menu Demo](demo.html)



\## ✨ Features



\- \*\*Pure CSS\*\* — No JavaScript required

\- \*\*Typewriter Animation\*\* — Animated text overlay on hover/focus

\- \*\*Fully Accessible\*\* — Uses native `<select>` element with proper ARIA attributes

\- \*\*Responsive\*\* — Adapts to all screen sizes

\- \*\*Customizable\*\* — Uses CSS variables for easy theming

\- \*\*Reduced Motion Support\*\* — Respects `prefers-reduced-motion`

\- \*\*Multiple Variants\*\* — Light, Dark, and Gradient themes included
