EaseMotion CSS

Copy-to-Clipboard Code Blocks

Documentation-style code blocks with copy-to-clipboard functionality.

Getting Started

Install the package using npm:

npm install easemotion-css

Or using yarn:

yarn add easemotion-css

Basic Usage

Import and use the animation classes:

<div class="ease-fade-in">
  Content
</div>
import { init } from 'easemotion-css';

init();

Configuration

Customize animations with CSS custom properties:

:root {
  --ease-duration: 500ms;
  --ease-easing: ease-out;
}
