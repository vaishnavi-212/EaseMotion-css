# Pulse Animation Example

A reusable Pulse Animation example for the EaseMotion CSS Standard Examples collection.

## Features

- Pure HTML & CSS
- Responsive layout
- Pulse animation
- Reusable UI components
- No JavaScript required

## Components Included

- Button
- Card
- Badge
- Notification
- Icon

## Folder Structure

```
submissions/examples/pulse-sp/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any modern browser.

Customize colors, animation duration, and spacing through CSS variables in `style.css`.

## Browser Support

- Chrome
- Firefox
- Safari
- Edge
