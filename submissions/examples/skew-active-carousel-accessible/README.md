# CSS Skew Active Carousel (Accessible)

A responsive and accessible portfolio carousel built entirely with HTML and CSS.

## Features

- Pure HTML & CSS
- Responsive design
- Smooth skew-active animation
- Keyboard accessible using `:focus-visible`
- CSS custom properties
- No JavaScript required

## Folder Structure

```text
examples/
└── skew-active-carousel-accessible/
    ├── demo.html
    ├── style.css
    └── README.md
```

## Customization

```css
:root{
  --accent:#2563eb;
  --skew:8deg;
  --duration:.45s;
  --active-scale:1.05;
}
```

## Accessibility

- Keyboard focus support
- Visible focus indicator
- Semantic HTML elements
- Responsive layout
