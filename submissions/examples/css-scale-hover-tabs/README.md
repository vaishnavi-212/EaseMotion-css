# CSS Scale Hover Tabs

A responsive **Scale Hover Tabs** component built using pure HTML and CSS for creative portfolio layouts.

## Features

- Pure HTML & CSS
- Scale hover animation
- Smooth transitions
- Responsive design
- No JavaScript
- Accessible with `prefers-reduced-motion`

## Files

- demo.html
- style.css
- README.md

## Usage

```html
<div class="tabs">
    <!-- Tab Buttons -->
    <!-- Tab Content -->
</div>
```

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Accessibility

Supports the `prefers-reduced-motion` media query for users who prefer reduced animations.
