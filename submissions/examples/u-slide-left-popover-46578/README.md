# CSS Slide-Left Popover for Product Catalog Layouts

A **pure CSS Slide-Left Popover** designed for **Product Catalog Layouts**. The component features a smooth slide-left animation, modern product showcase styling, responsive behavior, keyboard accessibility, and customizable CSS variables without JavaScript.

## Features

- Pure HTML & CSS
- Smooth Slide-Left animation
- Product Catalog inspired design
- Fully responsive
- Keyboard accessible (`:focus-within`)
- CSS Custom Properties
- Lightweight
- No JavaScript required
