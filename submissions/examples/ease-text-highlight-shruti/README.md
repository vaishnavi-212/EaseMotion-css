# ease-text-highlight-marker

A reusable text highlight component that animates like a marker swipe using pure CSS.

## Features

- Pure CSS
- Responsive
- Keyboard accessible
- Smooth marker highlight animation
- Uses CSS custom properties
- Lightweight and reusable

## Files

```
demo.html
style.css
README.md
```

## Custom Properties

```css
--ease-highlight-color
--ease-text-color
--ease-duration
--ease-easing
--ease-padding
--ease-radius
```

## Accessibility

- Keyboard focus support
- Responsive typography
- No JavaScript required
