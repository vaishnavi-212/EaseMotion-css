Accessibility: The trigger wrapper uses tabindex="0" and :focus-within, so the tooltip is reachable by keyboard. role="tooltip" is set on each bubble, and prefers-reduced-motion disables the lift for users who request reduced motion. Colors meet WCAG AA contrast ratios.

Parameter Controls

These sliders mutate the CSS custom properties that drive the fade-out transition. Changes apply live to every tooltip on the page.
