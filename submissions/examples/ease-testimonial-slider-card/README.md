# Ease Testimonial Slider Card

## Overview

A reusable testimonial slider card component built with pure CSS.

It provides a clean testimonial UI pattern with smooth entrance animation,
responsive design, and customizable styling.

## Features

- Pure CSS component
- Smooth slide entrance animation
- Responsive layout
- Modern card design
- Accessible buttons
- CSS variable customization
- No JavaScript dependency

## Usage

```html
<div class="testimonial-card">

<img class="avatar">

<p class="quote">
Your testimonial text
</p>

<h3>
User Name
</h3>

<span>
Role
</span>

</div>
