# Ease Temporary File Cleaner

## Overview

A modern CSS-based temporary file cleaner card component that displays storage cleanup information with smooth animations.

This component is designed for dashboards, system utilities, and SaaS interfaces.

---

## Features

- Pure CSS animation
- Responsive card layout
- Neumorphic style design
- Animated cleaning progress bar
- Clean and reusable UI
- No JavaScript dependency

---

## Usage

Include the component HTML:

```html
<div class="cleaner-card">
  <h2>Temporary File Cleaner</h2>
  <button class="clean-btn">
    Clean Now
  </button>
</div>
