# CSS Micro Shake Popover

A responsive **Micro Shake Popover** designed for **Marketing Landing Page Layouts** using only HTML and CSS.

## Features

- Pure HTML & CSS
- Responsive Design
- Keyboard Accessible (`:focus-visible`)
- Smooth Micro Shake Animation
- Hover & Focus Popover
- Customizable CSS Variables
- No JavaScript

## Technologies

- HTML5
- CSS3

## CSS Variables

```css
--primary
--secondary
--shake-duration
--shake-distance
--popover-bg
--shadow
--transition
```
