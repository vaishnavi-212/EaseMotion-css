# 🔀 Scramble Button – Text Scramble Effect

> A dynamic button where text scrambles and unscrambles on hover, creating a glitch-like, cyberpunk effect that captivates attention.

---

## 📖 Description

The **Scramble Button** is an engaging UI component that creates a dynamic text scrambling effect on hover. The button's text transforms into random characters and gradually reveals itself, creating a glitch-like, cyberpunk aesthetic that's both captivating and memorable. The effect is achieved using JavaScript-driven character substitution with smooth easing, making the scramble feel organic and satisfying.

The component features a dual-state interaction – on hover, the text scrambles into random characters and then unscrambles back to the original text. A subtle icon rotation and glitch animation enhance the effect, creating a cohesive, polished experience. This is perfect for buttons that need to stand out, convey a sense of technology, or simply add a moment of delight to routine interactions.
