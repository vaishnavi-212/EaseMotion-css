# CSS Zoom-Out Popover for Responsive Dashboard Layouts

A **pure CSS Zoom-Out Popover** built for **Responsive Dashboard Layouts**. The component features a smooth zoom-out animation, responsive dashboard styling, keyboard accessibility, and customizable CSS variables without using JavaScript.

## Features

- Pure HTML & CSS
- Smooth Zoom-Out animation
- Responsive Dashboard inspired design
- Fully responsive
- Keyboard accessible (`:focus-within`)
- CSS Custom Properties
- Lightweight
- No JavaScript required
