# Calendar Date Flip

This example adds a calendar event card with a date tile that flips into place.

## Files

- `demo.html` contains the event card and date tile markup.
- `style.css` defines the flip animation, calendar styling, and reduced-motion fallback.

## Highlights

- Uses 3D transform perspective for a date flip effect.
- Keeps the tile dimensions fixed for a stable layout.
- Includes readable event details below the animated date.
- Provides a no-motion fallback for reduced-motion users.
