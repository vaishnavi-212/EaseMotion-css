# Ease Notification Toast Stack

A modern CSS-only stacked notification toast component for EaseMotion CSS.

## Features

- 📢 Stacked notification layout
- ✅ Success, Warning & Error variants
- 🎨 Pure HTML & CSS
- ✨ Smooth slide-in animation
- 📱 Responsive design
- 🚫 No JavaScript required

## Folder Structure

```
ease-notification-toast-stack/
├── demo.html
├── style.css
└── README.md
```

## Usage

Include the stylesheet:

```html
<link rel="stylesheet" href="style.css">
```

Example:

```html
<div class="toast-stack">
    <div class="toast success">...</div>
    <div class="toast warning">...</div>
    <div class="toast error">...</div>
</div>
```

Open `demo.html` directly in your browser.
