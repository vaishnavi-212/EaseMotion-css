# Ease SSL Certificate Overview

## Overview
A CSS-only component for displaying SSL certificate validity and expiry information.

## Features
- Valid, warning, and expired status badges
- Responsive layout
- Smooth hover animation
- CSS variables
- Pure HTML & CSS

## Folder Structure
demo.html
style.css
README.md

## Usage
Open `demo.html` directly in your browser.

## Browser Support
Chrome, Firefox, Edge, Safari

## Responsive
Works on desktop, tablet, and mobile devices.

## Why EaseMotion CSS?
Provides a reusable animation-first component for security dashboards, hosting providers, DevOps platforms, and infrastructure monitoring tools.
