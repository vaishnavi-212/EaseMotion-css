# Print-Optimized Code Inline Component

Inline code styling with print media query optimization. Code renders with grayscale styling when printed.

## Files

- `demo.html` - Code inline demo with print optimization
- `style.css` - Code inline styling with @media print rules

## Usage

Open `demo.html` in a browser to see the inline code styling. Use Ctrl+P (Cmd+P on Mac) to see simplified print styling.

## What This Does

Adds print media query optimization to inline code components:
- Converts colored styling to grayscale
- Maintains readability on paper

## Why Useful

Colored code syntax highlighting wastes ink when printed. This optimization ensures code renders with clean, readable styling on paper.
