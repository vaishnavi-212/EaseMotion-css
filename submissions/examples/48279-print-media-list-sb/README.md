# Print-Optimized List Component

List components with print media query optimization. Lists render with borders and simplified styling when printed.

## Files

- `demo.html` - List demo with print optimization
- `style.css` - List styling with @media print rules

## Usage

Open `demo.html` in a browser to see the styled lists. Use Ctrl+P (Cmd+P on Mac) to see simplified print styling.

## What This Does

Adds print media query optimization to list components:
- Removes background colors
- Adds borders for readability
- Shows all items as plain text

## Why Useful

Lists with colored backgrounds and hover effects don't print well. This optimization ensures lists render as clean, readable data on paper.
