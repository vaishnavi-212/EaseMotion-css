# Table Row Expand

This example adds a table-like row that reveals supporting details on hover or keyboard focus.

## Files

- `demo.html` contains the deployment row and detail panel markup.
- `style.css` defines the row layout, detail expansion, responsive behavior, and reduced-motion fallback.

## Highlights

- Uses a real button for the interactive row.
- Keeps the row height stable and expands only the detail area.
- Supports hover and keyboard focus interactions.
- Removes transition movement for reduced-motion users.
