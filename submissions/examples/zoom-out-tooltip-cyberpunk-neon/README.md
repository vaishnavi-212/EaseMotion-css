# Zoom-Out Tooltip – Cyberpunk Neon Layout

**Issue:** #46248

## Description

A pure CSS **Zoom-Out Tooltip** featuring a Cyberpunk Neon theme with glowing effects and a smooth zoom-out animation. Built using only HTML and CSS with no JavaScript.

## Features

- Pure HTML & CSS
- Cyberpunk Neon styling
- Smooth Zoom-Out animation
- Responsive design
- Keyboard accessible
- CSS Custom Properties
- Zero JavaScript

## Usage

```html
<div class="tooltip-wrapper">
    <button class="neon-btn">Hover Me</button>
    <span class="tooltip">System Online ⚡</span>
</div>
```

Include the stylesheet:

```html
<link rel="stylesheet" href="style.css">
```

## Customization

Modify these CSS variables:

```css
:root{
  --primary:#00f7ff;
  --secondary:#ff00c8;
  --duration:.35s;
  --scale:.75;
}
```
