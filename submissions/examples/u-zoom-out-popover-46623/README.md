# CSS Zoom-Out Popover for Dashboard Analytics Layouts

A **pure CSS Zoom-Out Popover** designed for **Dashboard Analytics Layouts**. It features a smooth zoom-out entrance animation, responsive layout, keyboard accessibility, and customizable CSS variables without requiring JavaScript.

## Features

- Pure HTML & CSS
- Smooth Zoom-Out animation
- Dashboard Analytics design
- Fully responsive
- Keyboard accessible (`:focus-within`)
- CSS Custom Properties
- Lightweight
- No JavaScript
