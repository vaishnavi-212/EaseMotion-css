# Floating Orbit Tooltip - Neumorphic Soft Layouts

A CSS tooltip component with floating orbit animation effect designed for neumorphic soft layouts.

## Files

- `demo.html` - Demo page showing neumorphic UI with tooltips
- `style.css` - Tooltip styles and animations

## Usage

Open `demo.html` in a browser and hover over interactive elements to see the floating orbit tooltip animation.

## Features

### Floating Orbit Animation
- Tooltip gently floats with subtle rotation
- Amber/gold gradient with shadow
- Smooth continuous animation loop

### Neumorphic Components
- Profile card with avatar
- Statistics boxes
- Action buttons (connect, message)
- Settings toggle switches

### Design System
- Soft shadow neumorphic style
- Light gray background
- Amber accent color

## Classes

| Class | Description |
|-------|-------------|
| `ease-floating-orbit-nm-sb` | Floating orbit tooltip for neumorphic |
| `ease-nm-icon-btn-sb` | Neumorphic icon button |
| `ease-nm-icon-btn-sm-sb` | Small neumorphic icon button |
| `ease-nm-btn-primary-sb` | Primary neumorphic button |
| `ease-nm-btn-secondary-sb` | Secondary neumorphic button |
| `ease-nm-toggle-sb` | Neumorphic toggle switch |
| `stat-box` | Statistics box |

## Usage

Add `data-tooltip="Your text"` attribute to any element with `ease-floating-orbit-nm-sb` class:

```html
<button class="ease-floating-orbit-nm-sb" data-tooltip="Edit profile">
  Edit
</button>
```
