# Card Component

A clean card component with hover effects, tags, and call-to-action button.

## Files

- `demo.html` - Card examples
- `style.css` - Full styling with animations

## Features

- Hover lift effect
- Tag badges
- Gradient icon area
- CTA button
- Responsive grid

## Design Philosophy

Modern, engaging cards with satisfying hover animations and clear visual hierarchy.
