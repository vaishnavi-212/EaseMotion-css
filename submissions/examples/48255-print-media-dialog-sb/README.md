# Print-Optimized Dialog Component

Dialog/modal component with print media query optimization. Dialogs are hidden when printing.

## Files

- `demo.html` - Dialog demo with print optimization
- `style.css` - Dialog styling with @media print rules

## Usage

Open `demo.html` in a browser to see the dialog. Use Ctrl+P (Cmd+P on Mac) to see dialog hidden in print mode.

## What This Does

Adds print media query optimization to dialog components, hiding them when printing. Dialogs are interactive elements that have no purpose in printed documents.

## Why Useful

Dialogs overlay the page and are designed for interactive use. They should not appear in print output.
