# Ease Job Queue Dashboard

## Overview
A CSS-only dashboard component for monitoring background job queues.

## Features
- Animated progress bars
- Responsive layout
- Pure HTML & CSS
- CSS variables
- Lightweight design

## Folder Structure
demo.html
style.css
README.md

## Usage
Open `demo.html` directly in any browser.

## Browser Support
Chrome, Firefox, Edge, Safari

## Responsive
Optimized for desktop, tablet, and mobile devices.

## Why EaseMotion CSS?
Provides a reusable animation-first dashboard component for queue monitoring, backend services, DevOps tools, and cloud infrastructure.
