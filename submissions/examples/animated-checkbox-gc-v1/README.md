# Animated Checkbox UI Example

## 1. What does this do?

This submission demonstrates custom animated checkboxes with smooth check mark animation, focus states, and color variants. It uses hidden input elements and styled labels with CSS `@keyframes` to create polished checkbox interactions.

---

## 2. How is it used?

Open `demo.html` directly in your browser. The example uses `style.css` for checkbox styling and animations. Copy the `.animated-checkbox` markup and CSS into your project to reuse the custom checkbox pattern.

---

## 3. Why is it useful?

Custom checkboxes with animation provide better visual feedback than default browser checkboxes. Smooth transitions and clear focus states improve accessibility while maintaining a cohesive design system aesthetic.
