# Table Sort Arrow Flip

A sortable table-header style button with an arrow that flips to show the active sort direction.

## Files

- `demo.html` - sort button markup with accessible label text.
- `style.css` - button lift, arrow flip, focus-visible state, and reduced-motion support.

## Highlights

- Uses a native button for sorting interaction.
- Keeps the arrow decorative with `aria-hidden`.
- Mirrors hover behavior on keyboard focus.
