# Slide-Up Popover – Interactive Interface

A responsive and accessible **pure CSS Slide-Up Popover** designed for interactive interface layouts.

## Features

- Pure CSS (No JavaScript)
- Smooth Slide-Up animation
- Responsive design
- Keyboard accessible
- Customizable using CSS custom properties
- Modern interactive interface styling

## Files

```
demo.html
style.css
README.md
```

## Customization

```css
:root {
  --ease-popover-duration: 0.35s;
  --ease-popover-easing: ease;
  --ease-popover-distance: 18px;
  --ease-popover-scale: 0.98;
}
```

## Accessibility

- Keyboard focus support
- Uses `role="tooltip"`
- Visible focus indicator
- Mobile responsive

## Usage

Hover or focus the button to reveal the Slide-Up popover.
