# Ease Thumbnail Gallery Grid

A responsive CSS-only thumbnail gallery grid with smooth hover scale easing.

## Usage
Link `ease-thumbnail-gallery-grid.css` and use the `.ease-thumbnail-grid` and `.ease-thumbnail-grid__item` classes as shown in `demo.html`.

## Features
- Responsive auto-fill grid layout
- Smooth hover scale transition using a custom cubic-bezier ease
- Respects `prefers-reduced-motion`
