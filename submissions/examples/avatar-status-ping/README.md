# Avatar Status Ping

This example adds a profile card with an online status dot that pings gently.

## Files

- `demo.html` contains the profile card and avatar markup.
- `style.css` defines the avatar, status indicator, ping animation, and reduced-motion fallback.

## Highlights

- Uses a stable circular avatar layout.
- Adds a clear online status signal.
- Keeps the ping animation outside the main content flow.
- Hides the ping effect for reduced-motion users while keeping the status dot visible.
