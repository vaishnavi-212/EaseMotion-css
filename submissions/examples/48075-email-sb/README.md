# Email Newsletter Subscription Component

A clean, modern email subscription form for newsletters with floating card design and animated interactions.

## Files

- `demo.html` - Newsletter subscription form
- `style.css` - Full styling with animations

## Features

- Pure CSS design
- Floating card with shadow
- Input focus effects
- Button hover animations
- Responsive layout

## Design Philosophy

Minimal, conversion-focused newsletter signup with elegant animations and clear call-to-action.
