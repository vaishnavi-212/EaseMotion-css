# Typewriter Carousel

A responsive carousel component with a typewriter text animation, designed for admin panels, dashboards, and analytics pages. It is built using pure HTML and CSS without requiring JavaScript.

---

## Features

- Pure HTML and CSS
- No JavaScript required
- Typewriter heading animation
- CSS-only carousel navigation
- Responsive layout
- Simple and reusable design
- Compatible with EaseMotion CSS utility classes

## How to Use

Add the component to your HTML file:

```html
<section class="dashboard-carousel ease-fade-in">

    <div class="carousel">

        <div class="slides">

            <article class="card">

                <span class="status">● Active</span>

                <h2 class="typewriter">
                    Sales Overview
                </h2>

                <p>
                    Revenue increased by 24% this month.
                </p>

            </article>

        </div>

    </div>

</section>
```

Open `demo.html` in any modern browser to view the component.

---

## Why This Component?

Admin dashboards often need a simple way to present important information. This component combines a CSS-only carousel with a typewriter animation to highlight key updates while keeping the interface clean and engaging. Since it doesn't rely on JavaScript, it is lightweight and easy to integrate into different projects.
