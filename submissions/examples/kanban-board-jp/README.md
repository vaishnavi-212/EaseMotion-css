# 📋 Interactive Project Kanban Board

A modern and responsive **Kanban Board Component** built using HTML and CSS for the EaseMotion CSS examples collection.

## ✨ Features

- Responsive Kanban board
- To Do, In Progress, Review and Done columns
- Animated task cards
- Priority badges
- Progress bars
- User avatars
- Summary statistics
- Modern hover effects
- Responsive layout
- CSS variables
- Lightweight implementation
