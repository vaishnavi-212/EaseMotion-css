# Age Calculator

## What does this do?
An age calculator that computes exact age in years, months, and days from a given date of birth, with live precise breakdown including hours, minutes, and seconds.

## How is it used?
Open demo.html in a browser. Select a date of birth using the date picker. The age is displayed instantly in large format blocks, with a precise breakdown below.

## Why is it useful?
This provides a clean age calculation component suitable for registration forms, profile pages, or birthday countdowns. It demonstrates date arithmetic and live DOM updates.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to use the calculator.
