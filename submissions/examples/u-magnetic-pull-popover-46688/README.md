# CSS Magnetic Pull Popover for Neumorphic Soft Layouts

A pure CSS **Magnetic Pull Popover** with a soft neumorphic design. The popover smoothly "pulls" toward the trigger element, creating a modern interaction without JavaScript.

## Features

- Pure HTML & CSS
- Magnetic Pull animation
- Neumorphic Soft UI
- Responsive layout
- Keyboard accessible
- CSS Custom Properties
- No JavaScript
- Lightweight & reusable
