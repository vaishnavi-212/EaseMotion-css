# Fade-In Popover – Minimalist Tech

A lightweight, responsive, and accessible pure CSS Fade-In Popover inspired by minimalist tech interfaces.

## Features

- Pure CSS (No JavaScript)
- Smooth Fade-In transition
- Responsive layout
- Keyboard accessible
- Customizable through CSS variables
- Uses EaseMotion-style CSS custom properties

## Files

```
demo.html
style.css
README.md
```

## Customization

```css
:root {
  --ease-popover-duration: 0.35s;
  --ease-popover-easing: ease;
  --ease-popover-scale: 0.97;
}
```

## Accessibility

- Keyboard focus support
- Uses `role="tooltip"`
- Visible focus indicator
- Responsive on smaller screens
