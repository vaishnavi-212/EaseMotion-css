# Text Selection Highlight Utility

A lightweight CSS utility that customizes text selection colors using the `::selection` pseudo-element.

## Features

- Blue selection utility
- Green selection utility
- Red selection utility
- Lightweight and reusable
- Easy to integrate

## Classes

- `.ease-selection-primary`
- `.ease-selection-success`
- `.ease-selection-danger`

## Usage

```html
<p class="ease-selection-primary">
  Select this text.
</p>
