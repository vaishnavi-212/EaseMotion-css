# CSS Micro Shake Popover

A responsive **Micro Shake Popover** for **SaaS Showcase Layouts**, built entirely with HTML and CSS.

## Features

- Pure HTML & CSS
- Micro Shake Animation
- SaaS Showcase Design
- Responsive Layout
- Keyboard Accessible
- CSS Custom Properties
- No JavaScript

## Technologies

- HTML5
- CSS3

## Folder Structure

```
micro-shake-saas-popover/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.
