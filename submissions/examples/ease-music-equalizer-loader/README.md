# Ease Music Equalizer Loader

A modern CSS-only animated music equalizer loader for the EaseMotion CSS collection.

## Features

- 🎵 Equalizer-style loading animation
- 🎨 Pure HTML & CSS
- ⚡ Lightweight
- 📱 Responsive
- 🚫 No JavaScript required
- ♻️ Easy to customize

## Folder Structure

```
ease-music-equalizer-loader/
├── demo.html
├── style.css
└── README.md
```

## Usage

Include the stylesheet:

```html
<link rel="stylesheet" href="style.css">
```

Use the component:

```html
<div class="equalizer">
    <span></span>
    <span></span>
    <span></span>
    <span></span>
    <span></span>
</div>
```

Open `demo.html` directly in any browser.

- Chrome
- Firefox
- Edge
- Safari
