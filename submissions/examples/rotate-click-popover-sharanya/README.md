# Rotate Click Popover Dashboard

A modern **CSS-only rotating popover** that displays a compact analytics dashboard. The popover smoothly rotates into view when the button receives focus, creating an elegant interactive UI without using JavaScript.

## Features

* Pure HTML & CSS
* Smooth `rotateX` animation
* Clean analytics dashboard layout
* Responsive design
* No JavaScript required

## Technologies Used

* HTML5
* CSS3

## How It Works

* Click the **Analytics** button to focus it.
* The dashboard popover rotates into view using CSS `transform` and `transition`.
* Clicking elsewhere removes the focus, causing the popover to rotate back and hide.
