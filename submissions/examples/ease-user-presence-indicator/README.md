# Ease User Presence Indicator

A clean CSS-only avatar presence indicator with animated online status and support for **Online**, **Away**, **Busy**, and **Offline** states.

## Features

- 👤 Avatar profile cards
- 🟢 Animated online pulse
- 🟡 Away status
- 🔴 Busy status
- ⚪ Offline status
- ✨ Hover animation
- 📱 Responsive layout
- 🎨 Pure HTML & CSS

## Folder Structure

```
ease-user-presence-indicator/
├── demo.html
├── style.css
└── README.md
```

## Usage

```html
<link rel="stylesheet" href="style.css">

<div class="user online">
    <img src="avatar.jpg" alt="User">
    <span class="status"></span>
    <h3>Alex</h3>
    <p>Online</p>
</div>
```

Replace the class with:

- `.online`
- `.away`
- `.busy`
- `.offline`

to display different presence states.

Open `demo.html` directly in your browser.
