# Horizontal Roll Tooltip – E-Commerce Checkout

A responsive E-Commerce Checkout tooltip featuring a smooth Horizontal Roll animation built entirely with HTML and CSS.

## Features

- Pure HTML & CSS
- Horizontal Roll animation
- E-Commerce Checkout inspired interface
- Responsive design
- Keyboard accessible
- CSS custom properties
- Zero JavaScript

## Files

```
demo.html
style.css
README.md
```

## Usage

Open `demo.html` directly in any modern browser.

Customize using:

```css
--primary
--duration
--radius
```
