# Skeleton Jello Animation

A reusable example demonstrating the `ease-jello` animation class applied to a Skeleton loading placeholder.

## Usage

Simply add the `ease-jello` class to any placeholder elements in your skeleton UI:

```html
<div class="skeleton-card">
  <div class="skeleton-avatar ease-jello"></div>
  <div class="skeleton-title ease-jello"></div>
</div>
```

## Why is it useful?

It provides an engaging, dynamic alternative to standard pulse or shimmer loading states, drawing users' attention while content is fetched. By adhering to the `prefers-reduced-motion` media query, this component ensures an accessible experience for users sensitive to motion.
