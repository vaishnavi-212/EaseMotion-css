# Resource Monitor Graph Component

A dark-themed system resource monitor dashboard showing CPU, memory, disk, and network metrics.

## Files

- `demo.html` - Monitor dashboard with metrics
- `style.css` - Full styling with animations

## Features

- Pure CSS implementation
- CPU, Memory, Disk, Network metrics
- Animated progress bars
- Real-time activity chart
- Dark theme design

## Design Philosophy

Clean, informative dashboard for monitoring system resources with color-coded metrics.
