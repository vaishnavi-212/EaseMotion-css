# Ease DNS Record Manager

## Overview
A CSS-only component for displaying DNS records and their current status.

## Features
- Animated status badges
- Responsive layout
- CSS variables
- Hover effects
- Pure HTML & CSS

## Folder Structure
demo.html
style.css
README.md

## Usage
Open `demo.html` directly in any browser.

## Browser Support
Chrome, Firefox, Edge, Safari.

## Responsive
Works across desktop, tablet, and mobile devices.

## Why EaseMotion CSS?
Provides a reusable animation-first component for hosting dashboards, DNS management panels, DevOps tools, and cloud administration platforms.
