# Animated Neumorphic Glowing Toggle

A modern, tactile 3D toggle switch using pure CSS neumorphism (soft UI) combined with a vibrant neon glow effect when active.

## Features
- **Neumorphism Design**: Uses precise inset and outset box-shadows to create a physical, extruded 3D look that blends seamlessly with the background.
- **Neon Active State**: Transforms from a soft grey physical slider into a glowing, vibrant neon button when clicked.
- **Spring Physics**: Utilizes a custom `cubic-bezier` transition to give the toggle switch a satisfying, bouncy, and physical snap when toggled.

## How to use

```html
<label class="ease-neumorphic-toggle">
    <input type="checkbox" class="ease-toggle-checkbox">
    <div class="ease-toggle-slider"></div>
</label>
```
