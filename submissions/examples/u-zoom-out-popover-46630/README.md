# CSS Zoom-Out Popover for Cyberpunk Neon Layouts

A **pure CSS Zoom-Out Popover** with a futuristic **Cyberpunk Neon** aesthetic. The component uses a smooth zoom-out entrance animation, responsive layout, keyboard accessibility, and customizable CSS variables without requiring JavaScript.

## Features

- Pure HTML & CSS
- Zoom-Out animation
- Cyberpunk Neon design
- Fully responsive
- Keyboard accessible (`:focus-within`)
- CSS Custom Properties
- Lightweight
- No JavaScript
