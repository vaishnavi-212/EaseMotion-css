# Elastic Bounce Popover – Creative Portfolio

A standalone responsive popover demonstrating an Elastic Bounce interaction for creative portfolio layouts using pure HTML and CSS.

## Features

- Pure HTML & CSS
- Elastic Bounce animation
- Creative portfolio UI
- Responsive layout
- Keyboard accessible
- Customizable CSS variables
- Zero JavaScript

## Files

```
demo.html
style.css
README.md
```

## Usage

Open `demo.html` directly in your browser.

Customize using CSS variables:

```css
--primary
--secondary
--duration
--radius
```
