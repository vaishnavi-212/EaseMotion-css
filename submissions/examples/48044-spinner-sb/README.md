# Spinner Component

A clean loading spinner component with multiple sizes and colors.

## Features

- Multiple sizes (sm, md, lg)
- Multiple colors (default, primary, success, warning)
- Smooth rotation animation

## Usage

```html
<div class="spinner"></div>
<div class="spinner spinner-sm spinner-primary"></div>
```

## CSS Variables

- `--ease-spinner-size`: Size of the spinner
- `--ease-spinner-color`: Color of the spinner
