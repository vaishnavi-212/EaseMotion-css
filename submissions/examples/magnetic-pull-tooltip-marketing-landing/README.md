# Magnetic Pull Tooltip – Marketing Landing Page

**Issue:** #46298

## Description

A pure CSS **Magnetic Pull Tooltip** designed for Marketing Landing Pages. The tooltip smoothly moves toward the trigger element, creating a subtle magnetic pull effect without JavaScript.

## Features

- Pure HTML & CSS
- Magnetic Pull animation
- Marketing Landing Page design
- Responsive layout
- Keyboard accessible
- CSS Custom Properties
- No JavaScript required

## Usage

```html
<div class="tooltip-container">
    <button class="cta-btn">Get Started</button>
    <span class="tooltip">Start your free trial today 🚀</span>
</div>
```

Include:

```html
<link rel="stylesheet" href="style.css">
```

## Customization

Modify CSS variables:

```css
:root{
    --primary:#6c63ff;
    --secondary:#8f94fb;
    --duration:.35s;
    --pull-distance:18px;
}
```
