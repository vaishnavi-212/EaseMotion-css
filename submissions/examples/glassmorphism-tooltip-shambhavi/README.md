# Glassmorphism Staggered Entrance Tooltip

A standalone responsive Glassmorphism Tooltip built using pure HTML and CSS.

## Features

- Pure HTML & CSS
- Staggered entrance animation
- Glassmorphism interface
- Responsive design
- Keyboard accessible
- CSS custom properties
- Zero JavaScript

## Files

```
demo.html
style.css
README.md
```

## Usage

Open `demo.html` in your browser.

Customize:

```css
--duration

--translate

--radius
```
