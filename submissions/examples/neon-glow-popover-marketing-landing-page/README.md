# Neon Glow Popover – Marketing Landing Page

**Issue:** #46481

## Overview

A pure CSS Neon Glow Popover designed for Marketing Landing Page interfaces. It features a modern glowing animation without JavaScript.

## Features

- Pure CSS
- Responsive Design
- Keyboard Accessible
- CSS Custom Properties
- Smooth Neon Glow Animation
- Lightweight

## CSS Variables

```css
:root{
  --glow-color:#00e5ff;
  --duration:.45s;
  --scale:.95;
}
```

## Usage

```html
<div class="popover">
    <button class="trigger">Learn More</button>

    <div class="popover-content">
        Marketing Content
    </div>
</div>
```
