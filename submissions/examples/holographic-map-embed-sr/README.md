# Holographic Map Embed

A minimalist **Holographic Map Embed** component built for the EaseMotion CSS library.

This component provides a modern map embedding experience with a glassmorphism UI, holographic scanning effect, smooth floating animation, and responsive layout using only HTML and CSS.

## ✨ Features

- 🌐 Interactive map embed using iframe
- 💎 Holographic scanning overlay effect
- 🪄 Smooth floating card animation
- 🟢 Animated status indicator
- 🎨 Uses EaseMotion CSS variables
- ⚡ Pure CSS implementation (No JavaScript)
- 📱 Fully responsive design
- ♿ Accessibility support with reduced motion preference
- 🧊 Glassmorphism and minimalist UI style
