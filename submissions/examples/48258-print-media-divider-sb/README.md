# Print-Optimized Divider Component

Divider components with print media query optimization. Dividers render with simple black borders when printed.

## Files

- `demo.html` - Divider demo with print optimization
- `style.css` - Divider styling with @media print rules

## Usage

Open `demo.html` in a browser to see the dividers. Use Ctrl+P (Cmd+P on Mac) to see simplified print styling.

## What This Does

Adds print media query optimization to divider components:
- Converts all divider styles to solid black
- Simplifies border width

## Why Useful

Colored and styled dividers don't print well. This optimization ensures dividers render as clean, readable separators.
