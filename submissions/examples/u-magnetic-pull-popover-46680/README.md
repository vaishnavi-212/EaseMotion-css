# CSS Magnetic Pull Popover for Marketing Landing Page Layouts

A pure CSS **Magnetic Pull Popover** created for modern marketing landing pages. The popover smoothly pulls toward the trigger element, highlighting promotional content without JavaScript.

## Features

- Pure HTML & CSS
- Magnetic Pull animation
- Marketing landing page design
- Fully responsive
- Keyboard accessible
- CSS Custom Properties
- Lightweight and reusable
- No JavaScript required
