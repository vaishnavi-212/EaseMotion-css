# Alert Component

A clean alert component with multiple variants and icons.

## Features

- Multiple variants (info, success, warning, error)
- Icon support
- Smooth slide-in animation
- Accessible color combinations

## Usage

```html
<div class="alert alert-info">
  <span class="alert-icon">ℹ</span>
  <div class="alert-content"><strong>Info:</strong> Message here.</div>
</div>
```

## CSS Classes

- `.alert` - Base alert styles
- `.alert-info` - Blue informational alert
- `.alert-success` - Green success alert
- `.alert-warning` - Yellow warning alert
- `.alert-error` - Red error alert
