# Staggered Entrance Tooltip – Accessible Web

A responsive Accessible Web tooltip featuring a smooth staggered entrance animation using only HTML and CSS.

## Features

- Pure HTML & CSS
- Staggered Entrance animation
- Accessible Web inspired UI
- Keyboard accessible
- Responsive design
- CSS custom properties
- Zero JavaScript

## Files

```

style.css
README.md
```

## Usage

Open `demo.html` directly in any browser.

Customize:

```css
--duration
--delay
--radius
```
