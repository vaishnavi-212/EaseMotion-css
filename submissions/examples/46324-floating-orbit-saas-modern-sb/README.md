# Floating Orbit Tooltip - Modern SaaS Layouts

A CSS tooltip component with floating orbit animation effect designed for modern SaaS interfaces.

## Files

- `demo.html` - Demo page showing modern SaaS UI with tooltips
- `style.css` - Tooltip styles and animations

## Usage

Open `demo.html` in a browser and hover over interactive elements to see the floating orbit tooltip animation.

## Features

### Floating Orbit Animation
- Tooltip gently floats with subtle rotation
- Teal/cyan gradient with shadow
- Smooth continuous animation loop

### Modern SaaS Components
- User profile header
- Quick stats cards
- Action tile grid
- Notification panel

### Design System
- Modern SaaS aesthetic
- Dark profile header
- Teal accent color
- Clean card-based layout

## Classes

| Class | Description |
|-------|-------------|
| `ease-floating-orbit-saas-m-sb` | Floating orbit tooltip for modern SaaS |
| `ease-profile-menu-sb` | Profile menu button |
| `ease-action-tile-sb` | Action tile button |
| `ease-mark-read-sb` | Mark all read button |

## Usage

Add `data-tooltip="Your text"` attribute to any element with `ease-floating-orbit-saas-m-sb` class:

```html
<button class="ease-floating-orbit-saas-m-sb" data-tooltip="Open menu">
  Menu
</button>
```
