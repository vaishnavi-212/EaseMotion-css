# Bounce Map Embed

A responsive SaaS-style animated map embed component for EaseMotion CSS.

## Features

- Responsive layout
- Pure CSS animation
- No JavaScript required
- Animated bouncing location pin
- Ripple effect
- Accessible iframe
- Mobile friendly
- Uses EaseMotion animation utilities

## Folder

```
bounce-map-embed-sr/
│
├── demo.html

```

## Usage

Open `demo.html` in a browser.

- Semantic HTML
