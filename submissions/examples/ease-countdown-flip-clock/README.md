# ease-countdown-flip-clock

**Issue:** #49949

A CSS-only flip-style countdown clock component featuring animated digit transitions.

---

## Features

- Flip animation effect
- Hours, Minutes, Seconds layout
- Pure HTML & CSS
- Responsive design
- No JavaScript required
- Easy customization
- Modern dark UI

---

## Folder Structure

```
ease-countdown-flip-clock/
├── demo.html
├── style.css
└── README.md
```

---

## Usage

```html
<link rel="stylesheet" href="style.css">

<div class="flip-clock">
    ...
</div>
```

Open `demo.html` in any browser.

---

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why it fits EaseMotion CSS

- Human-readable class names
