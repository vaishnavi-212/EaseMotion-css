# CSS Neon Glow Popover (Keyboard Accessible)

A modern, high-visibility neon overlay popover engineered completely without JavaScript framework logic. It uses modern keyboard accessibility states alongside CSS transform filters to scale fluidly.

## Key Features

- **Pure CSS `:focus-within` Mechanics:** Remains responsive to standard mouse movements while ensuring screen readers and keyboard users navigating via `Tab` trigger full display logic.
- **Hardware Scale Matrix:** Utilizes custom-bound curves (`--popover-easing`) to run acceleration metrics directly off the compositor layout pipeline.
- **Neon Diffuse Ambience:** Merges multi-layered linear configurations and webkit vector clipping masks to simulate crisp ambient lighting properties over active states.
