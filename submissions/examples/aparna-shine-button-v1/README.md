# Shine Button Utility

A reusable CSS utility that creates a smooth shine animation across buttons on hover.

## Features

- Smooth shine effect
- Lightweight and reusable
- Pure HTML & CSS
- Easy to customize
- Suitable for CTA buttons

## Usage

```html
<button class="shine-button">
  Get Started
</button>
```
