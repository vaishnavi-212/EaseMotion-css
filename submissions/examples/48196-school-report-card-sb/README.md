# School Report Card Component

A beautifully designed, pure CSS school report card component for educational websites and student management systems.

## Files

- `demo.html` - Complete report card markup with student details, grades table, and summary
- `style.css` - Full styling including animations, responsive design, and grade badges

## Usage

Open `demo.html` in any modern browser to view the animated report card component.

### HTML Classes

| Class | Description |
|-------|-------------|
| `.report-card` | Main container for the report card |
| `.card-header` | School branding and header section |
| `.student-details` | Grid layout for student information |
| `.grades-table` | Academic performance table |
| `.grade-badge` | Grade indicator badges (A, B, C) |
| `.summary-section` | Total marks, percentage, and overall grade |
| `.remarks-section` | Teacher's remarks area |
| `.card-footer` | Signature areas for teacher and principal |

### CSS Custom Properties

| Property | Default | Description |
|----------|---------|-------------|
| `--primary-color` | #4f46e5 | Main brand color |
| `--success-color` | #059669 | Good grade color |
| `--warning-color` | #d97706 | Average grade color |
| `--danger-color` | #dc2626 | Poor grade color |

## Features

- **Pure CSS implementation** - No JavaScript required
- **Smooth animations** - Card entrance, row slide-in, floating logo effects
- **Responsive design** - Adapts to mobile, tablet, and desktop
- **Grade badges** - Color-coded A/B/C performance indicators
- **Accessibility** - Uses semantic HTML and ARIA labels

## Design Philosophy

This component follows EaseMotion's philosophy of combining visual appeal with functional clarity. The report card uses subtle animations to guide attention while maintaining professional, trustworthy aesthetics suitable for educational contexts.
