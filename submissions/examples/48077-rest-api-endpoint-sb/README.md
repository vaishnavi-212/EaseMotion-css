# REST API Endpoint Card Component

A clean API documentation card component for displaying REST API endpoints with method badges, paths, and status codes.

## Files

- `demo.html` - API endpoint documentation
- `style.css` - Full styling

## Features

- Color-coded HTTP method badges (GET, POST, PUT, DELETE)
- Status code indicators
- Monospace font for paths
- Hover effects on cards

## Design Philosophy

Clean, developer-friendly API documentation cards that make it easy to scan and understand endpoints.
