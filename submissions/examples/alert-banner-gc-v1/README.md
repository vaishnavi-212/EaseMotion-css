# Alert Banner UI Example

## 1. What does this do?

This submission demonstrates a CSS-first alert banner component with success, warning, error, and info variants. It includes icon styling, dismiss buttons, and a slide-out dismiss animation using CSS `@keyframes`.

---

## 2. How is it used?

Open `demo.html` directly in your browser. The showcase includes `style.css` for visual styling and an inline script to toggle alert dismissal. Copy the alert markup and CSS into your project to use the `.alert-banner` class and type-specific modifiers such as `.alert-banner--success`.

---

## 3. Why is it useful?

Alert banners are essential for surface-level notifications, warnings, and success messages. This example provides reusable UI patterns with visual hierarchy, iconography, and smooth dismiss animations that improve user feedback and make the interface feel polished.
