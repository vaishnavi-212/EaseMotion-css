# Shimmer Sweep Tooltip - Neumorphic Soft Layouts

A CSS tooltip component with elegant shimmer sweep animation designed for neumorphic soft layouts.

## Files

- `demo.html` - Demo page showing all tooltip variants
- `style.css` - Tooltip styles and animations

## Usage

Open `demo.html` in a browser and hover over the buttons to see the shimmer sweep tooltip animation.

## Features

### Shimmer Sweep Animation
- Smooth gradient sweep effect on tooltip background
- Animated arrow indicator
- Scale entrance animation

### Neumorphic Design
- Soft shadow effects for depth
- Subtle color palette
- Rounded corners throughout

### Components
- Neumorphic buttons with tooltips
- Interactive card with refresh action
- Form controls with clear button

## Classes

| Class | Description |
|-------|-------------|
| `ease-shimmer-tooltip-sb` | Adds shimmer sweep tooltip to element |
| `ease-btn-neumorphic-sb` | Neumorphic styled button |
| `ease-btn-icon-sb` | Small neumorphic icon button |
| `neumorphic-card-sb` | Neumorphic card container |
| `ease-input-neumorphic-sb` | Neumorphic styled input |

## Usage

Add `data-tooltip="Your text"` attribute to any element with `ease-shimmer-tooltip-sb` class:

```html
<button class="ease-shimmer-tooltip-sb" data-tooltip="Save">
  <span>Icon</span>
</button>
```
