Copy Class Name button

A small copy icon next to each class name in the animation gallery — click it to copy that exact class straight to your clipboard, instead of manually selecting the text.

This mirrors the "Motion Library" card pattern from EaseMotion's own demo gallery (click a glyph to replay its animation, class name shown underneath). Since the real gallery lives outside submissions/, this is a self-contained, reusable version of just the copy-button pattern the issue asks for. It's a close cousin of the copy-code-btn-hr18 component submitted for issue #48659 — that one copies a whole code block, this one copies a single class name with the exact "Class name copied!" message the issue specifies.
