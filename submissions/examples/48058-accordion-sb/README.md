# Accordion Component

A clean accordion component with smooth expand/collapse animations and plus/minus toggle icon.

## Files

- `demo.html` - Accordion examples
- `style.css` - Full styling with animations

## Features

- Smooth expand/collapse animation
- Plus/minus toggle icon
- Clean hover states
- Accessible accordion

## Design Philosophy

Functional accordion with satisfying animations for expanding and collapsing content sections.
