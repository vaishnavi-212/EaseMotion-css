# Zoom-Out Tooltip – Responsive Dashboard

A responsive dashboard tooltip demonstrating a smooth **Zoom-Out** interaction using only HTML and CSS.

## Features

- Pure HTML & CSS
- Zoom-Out tooltip animation
- Responsive Dashboard inspired design
- Keyboard accessible
- Fully responsive
- CSS custom properties
- Zero JavaScript

## Files

Open `demo.html` directly in your browser.
