# BrightSmile Dental — Trusted Family Dentist Landing Page

1. **What does this do?**  
   A clean, friendly full-page business landing page for a dental clinic, built entirely with EaseMotion CSS utility classes and component classes — no custom keyframes.

2. **How is it used?**  
   Open `demo.html` directly in any browser. The page links to EaseMotion CSS via CDN and uses only existing `ease-*` classes for all animations, layout, and interactions.

3. **Why is it useful?**  
   Demonstrates how EaseMotion CSS can build complete, production-quality healthcare business landing pages with zero custom animations and no build step — just utility classes and components. This dental theme uses clean blue/teal tones for a trustworthy healthcare aesthetic.

## Sections

| Section | Description |
|---------|-------------|
| Hero | Full-viewport hero with blue-teal gradient, floating circle animation (`ease-float`), entrance animations (`ease-slide-up`, `ease-delay-*`) |
| Services | 6 dental service cards with icons, descriptions, and `ease-card-lift` hover |
| Smile Gallery | 3 before/after transformation cards with side-by-side comparison tiles |
| Doctor Profiles | 3 dentist profile cards with avatar initials, credentials, and `ease-card-lift` |
| Insurance & Pricing | 3-tier pricing cards + accepted insurance badge grid |
| Testimonials | 3 patient review cards with avatar initials and star ratings |
| Appointment Form | 2-column booking form with name, email, phone, date/time pickers, reason dropdown, and contact info sidebar |
| Footer | 4-column footer with brand, quick links, contact, and hours |
