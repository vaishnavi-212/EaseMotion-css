Theme Toggle Icon Fix

Issue #47290 — Moon icon visibility in light mode

Problem

In light mode, the moon icon (theme toggle) remains dark, making it hard to see against the light navigation bar background.

Fix Applied

Click the toggle button above to switch themes. The icon now adapts to the active theme:

  • Dark mode: Sun icon visible (white/light)
  • Light mode: Moon icon visible (dark)

CSS Change

/* Add to docs.css after line 252 */

[data-theme="light"] .theme-toggle-btn .moon-icon {
  opacity: 1;
  transform: rotate(0deg) scale(1);
  color: #1e293b;  /* dark color for light mode visibility */
}

[data-theme="dark"] .theme-toggle-btn .moon-icon {
  color: #e2e8f0;  /* light color for dark mode */
}
