Modal Dead Code Fix

Demonstration of the fix for issue #44509 — removing unreachable dead code in core/modal.js.

The Bug

In core/modal.js, the checkModal function contains a redundant conditional check:

// Line 24: class is added
overlay.classList.add('is-active');

const modal = overlay.querySelector('.ease-modal');
if (modal) {
  // Line 28: This check ALWAYS evaluates to false
  // because 'is-active' was just added on line 24
  if (!overlay.classList.contains('is-active')) {
    previousFocusedElement = document.activeElement; // DEAD CODE
  }

  modal.setAttribute('tabindex', '-1');
  modal.focus();
}

The Fix

Remove the unreachable if block. previousFocusedElement is already captured on line 22.

// Line 22: already captured
previousFocusedElement = document.activeElement;

overlay.classList.add('is-active');

const modal = overlay.querySelector('.ease-modal');
if (modal) {
  // Dead code removed — clean and direct
  modal.setAttribute('tabindex', '-1');
  modal.focus();
}
