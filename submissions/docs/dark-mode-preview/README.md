# Dark Mode Preview Examples

## Overview

This contribution adds dark mode preview examples for EaseMotion animations.

The goal is to help developers understand how animations appear when used with dark backgrounds.

## Added Features

- Dark theme preview page
- Fade animation example
- Floating animation example
- Pulse animation example
- Documentation for dark background usage

## How To Run

1. Open `demo.html` in a browser.
2. View the animations on the dark theme.

## Benefits

Dark mode previews improve developer experience by showing how animations behave in different UI themes.
