ease-toast

A toast notification component with slide-in animation, auto-dismiss, stack layout, and 4 semantic variants.

Triggers

Spawn Toasts

Click any button to spawn a toast. Use Auto-dismiss to see auto-removal after 4 seconds.

.ease-toast-{variant}

Usage

HTML Structure

Use the stack container for auto-positioned toast trays, or place individual toasts anywhere.

<!-- Stack container (auto-positioned)-->
<div class="ease-toast-stack ease-toast-stack-top-right">
  <div class="ease-toast ease-toast-success">
    <span class="ease-toast-icon">...</span>
    <div class="ease-toast-body">
      <strong>Title</strong>
      <p>Message</p>
    </div>
    <button class="ease-toast-close">&times;</button>
  </div>
</div>

<!-- Auto-dismiss (fades out after 4s) -->
<div class="ease-toast ease-toast-success ease-toast-auto-dismiss">
  ...
</div>
